# Обзор системы (SYSTEM.md)

## Концепция
Telegram MIDI Studio — это генеративная музыкальная среда, оптимизированная для работы внутри Telegram Mini Apps (TMA). Система построена на принципе "генеративного сотрудничества", где алгоритмы (Sting, Turing Machine, Bjorklund) создают паттерны, а пользователь управляет их развитием в реальном времени.

## Архитектурные слои

### 1. Слой данных и состояния (Zustand)
Проект использует **Zustand** для управления состоянием. Состояние разделено на логические домены:
- **`audioStore`**: Глобальный звуковой движок. Управляет `Tone.Transport`, инициализацией аудио-контекста (важно для iOS), BPM, Swing и мастеринг-цепочкой (Limiter).
- **`instrumentStore`**: Хранит параметры всех синтезаторов (Acid, FM, Harm) и секвенсоров. 
- **`drumStore`**: Специализированное хранилище для драм-машины (эвклидовы параметры, сэмплы).
- **`harmonyStore`**: Глобальный источник истины для тональности (Root) и лада (Scale). Все генераторы подписываются на изменения здесь.

### 2. Аудио-движок (Tone.js)
Движок инициализируется лениво (`lazy initialization`) при первом взаимодействии пользователя ("Start Studio").
- **Сигнальная цепочка**: Каждый инструмент имеет свою цепочку эффектов (Distortion, Filter, Delay), завершающуюся на глобальной шине.
- **Полифония**: `HarmSynth` использует Voice Pool для эффективного управления голосами.

### 3. Глобальный цикл (Sequencer Loop)
Центральный `Tone.Loop` (16-е ноты) — это сердце системы. Он:
1. Опрашивает генераторы.
2. Проверяет активные шаги.
3. Применяет "Smart Chord" трансформацию для Lead.
4. Отправляет MIDI-события в синтезаторы и UI-монитор.

## MIDI Экспорт
Реализован через `MidiExporter.ts` и хук `useMidiExport`. Поддерживает многодорожечную запись (Drums, Bass, Harm, Pads). Результат конвертируется в Blob и отправляется через Telegram WebApp API как файл.

## Безопасность и ограничения
- **iOS Audio**: Реализован воркараунд с `Tone.start()` внутри `onClick`.
- **Производительность**: Использование `Tone.Draw.schedule` для синхронизации UI с аудио-событиями без блокировки основного потока.
