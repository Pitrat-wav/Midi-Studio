# Навык: React Patterns

## 🧩 Компонентный Подход
- **Чистота**: Компоненты должны быть максимально чистыми. Логика — в хуках или `logic/`.
- **R3F**: Помни, что `<mesh>` — это тоже React-компонент. Используй `useFrame` вместо `useEffect` для анимаций.

## 🎣 Custom Hooks
- Выноси любую сложную логику в `src/hooks/`.
- Пример: `useMidiExport.ts` инкапсулирует всю сложность генерации файла.

## 🚫 Анти-паттерны
- Не используй `jQuery` или прямые манипуляции DOM (кроме Canvas).
- Не используй `useEffect` для анимации (будет лагать). Используй `Ref` + `useFrame`.
- Не пиши инлайн-стили, если можно использовать CSS-переменные темы.
