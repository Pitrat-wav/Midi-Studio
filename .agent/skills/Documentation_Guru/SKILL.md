# Навык: Гуру Документации

## 📚 Правило Синхронизации
- **Code First, Docs Immediately**: Изменил код -> Обновил доку.
- **Язык**: Вся документация (`docs/`) должна быть на **РУССКОМ** языке.
- **Источники**: Основной источник правды — `.agent/STUDIO_BIBLE_RU.md`.

## 🗂 Структура
- `docs/manual/`: Инструкции для ЧЕЛОВЕКА (юзера).
- `docs/technical/`: Инструкции для РАЗРАБОТЧИКА (архитектура, алгоритмы).
- `.agent/`: Инструкции для АГЕНТА.

## 📝 Changelog
- Всегда обновляй `CHANGELOG.md` при крупных изменениях. Формат: `## vX.Y.Z - Title (Date)`.
