# Навык: Git и Деплой

## 📍 Чекпоинты
- Создавай коммит после каждого крупного, рабочего изменения.
- Не пушь сломанный код.

## 📦 Versioning
- Используй **Semantic Versioning** (v5.0.0).
- Major: Ломающие изменения архитектуры.
- Minor: Новые фичи (Gamepad, Themes).
- Patch: Багфиксы.

## ☁️ Deployment
- Проект готов к деплою на Vercel (Frontend) и Railway (Backend).
- **Environment**: Проверяй `.env` на наличие API ключей перед билдом.
