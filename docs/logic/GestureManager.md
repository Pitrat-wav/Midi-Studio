# 📘 АУДИТ: GestureManager.ts

## 1. Глобальное Описание

### Назначение
Этот файл реализует систему распознавания жестов для управления 3D-студией через тачскрин или мышь. **GestureManager** позволяет приложению отличать простое нажатие (Tap) от смахивания (Swipe) для навигации или перетаскивания (Drag) для настройки параметров синтезатора. Также он реализует логику "долгого нажатия" (Hold) для открытия радиального меню.

### Связи
- **Store:** Использует собственное хранилище `useGestureStore` (Zustand).
- **Платформа:** Интегрирован с Telegram WebApp API для тактильной отдачи (Haptic Feedback).
- **Использование:** Служит входным фильтром для всех 3D-контроллеров и камер.

---

## 2. Интерфейс и UI Элементы

Этот модуль управляет логикой активации следующих элементов:
- **Радиальное меню (Radial Menu):** Открывается при удержании пальца в одной точке экрана более 600мс.
- **Управление камерой:** Переключение между видами (3D, Nodes, Live) через смахивание от краев экрана.
- **Ручки и слайдеры (Knobs/Sliders):** Активируются в режиме `drag`, позволяя менять значения параметров.

## 3. Логика и Математика

### 3.1 Определение начальной зоны (Edge Detection)

**Строки:** 42-58

При первом касании система проверяет, находится ли палец в "чувствительной зоне" у края экрана (порог — 60 пикселей). Это критично для отличия навигационных жестов (назад в Telegram) от управления внутри программы.

---

### 3.2 Математика жеста Swipe vs Drag

**Строки:** 75-91

Для определения того, что пользователь начал движение, используется Евклидово расстояние:
$$Distance = \sqrt{(x - x_{start})^2 + (y - y_{start})^2}$$

Если дистанция превышает 20 пикселей, система переходит в активное состояние:
- Если касание началось у края → `swipe`.
- Если под пальцем находится 3D объект (`targetPosition`) → `drag`.

---

### 3.3 Логика долгого нажатия (Long-Press / Hold)

**Строки:** 60-72

Реализована через таймер (`setTimeout` на 600мс). Чтобы предотвратить ложное срабатывание при дрожании руки, проверяется "радиус покоя":

$$ \Delta x < 10 \text{ и } \Delta y < 10 $$

Если за 600мс палец сместился менее чем на 10 пикселей, активируется жест `hold` и вызывается вибрация `medium` через Telegram Haptic API.
