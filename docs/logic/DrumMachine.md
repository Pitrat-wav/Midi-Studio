# 📘 АУДИТ: DrumMachine.ts

## 1. Глобальное Описание

### Назначение
Этот файл реализует класс `DrumMachine`, который отвечает за синтез ударных инструментов. В отличие от сэмплерных драм-машин, здесь звук генерируется математически в реальном времени с использованием специализированных синтезаторов Tone.js (MembraneSynth для бочки, NoiseSynth для малого барабана и т.д.).

### Связи
- **Импорты:** `tone` — аудио-движок.
- **Используется в:** `src/store/audioStore.ts` (глобальное управление звуком) и `src/components/WebGL/instruments/DrumMachine3D.tsx` (визуальные 3D пады).

## 2. Интерфейс и UI Элементы

❌ **Этот файл не содержит прямых UI-элементов.**

- `setKit` → Вызывается при переключении режима (808 / 909).

## 3. Логика и Математика

### 3.1 Конструктор и Архитектура Цепи

**Строки:** 13-54

**Что делает:**  
Создает независимые модули синтеза для каждого типа барабана и объединяет их в общую шину с компрессором.

**Математическая модель (Audio Flow):**
$$Signal_{Total} = Compressor(Kick + Snare + HiHat + Clap + Ride)$$

- **Kick (MembraneSynth):** Использует синусоидальную волну с быстрым спадом частоты.
- **Snare (NoiseSynth):** Генерирует белый шум с резкой огибающей амплитуды.
- **Ride (MetalSynth):** Использует аддитивный синтез с частотой гармоник 5.1 для металлического звона.

---

### 3.2 Функция: `setKit(kit)`

**Строки:** 60-91

**Что делает:**  
Переключает звуковую эстетику между классическими эмуляциями Roland TR-808 и TR-909.

**Логические шаги:**
- **Mode 808:** Длинные "хвосты" (decay), чистый синус на бочке, белый шум на малом барабане.
- **Mode 909:** Более короткие, "панчевые" звуки. Бочка имеет меньше октав спада частоты. Малый барабан использует розовый шум (Pink Noise) для более плотного тела звука.

---

### 3.3 Функция: `triggerDrum(drum, time, velocity)`

**Строки:** 114-121

**Что делает:**  
Запускает воспроизведение конкретного инструмента в указанный момент времени.

**Построчный разбор:**
1. Определяет тип инструмента.
2. Для каждого типа используется своя нота или тип шума:
   - `kick` -> Нота `C1` на 1/8 долю.
   - `hihat` -> Белый шум на сверхкороткую 1/32 долю.
   - `hihatOpen` -> Шум на 1/16 долю.
3. Отправляет команду в `triggerAttackRelease` соответствующего синта.
