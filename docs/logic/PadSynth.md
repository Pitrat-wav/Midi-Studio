# 📘 АУДИТ: PadSynth.ts

## 1. Глобальное Описание

### Назначение
Этот файл реализует **Pad Synth** — полифонический синтезатор, предназначенный для создания мягких, обволакивающих "подкладок" (пэдов). Он оптимизирован для игры аккордами и создания атмосферного фона. Главная особенность этого инструмента — использование расстроенных (detuned) осцилляторов и очень длинных огибающих для достижения плавности звука.

### Связи
- **Импорты:** `tone` (использует `Tone.PolySynth`).
- **Используется в:** `src/store/audioStore.ts` и управляется в `src/components/HUD/PadScreen.tsx`.
- **Влияние:** Часто используется вместе с `PadGenerator.ts`, который подает на вход этому синтезатору аккорды.

---

## 2. Интерфейс и UI Элементы

Синтезатор пэдов управляется через следующие экранные параметры:
- **Brightness (Яркость):** Управляет частотой среза фильтра. Чем выше значение, тем более открытым и "воздушным" становится звук.

## 3. Логика и Математика

### 3.1 Архитектура звука (Fat Sawtooth)

**Строки:** 20-33

Для создания объемного звука используется тип осциллятора `fatsawtooth`. Это не одна волна, а стек из нескольких ("count: 3"), которые слегка расстроены друг относительно друга ("spread: 30").

**Математическая модель:**
$$f_{osc\_i} = f_{base} + \text{detune}_i$$
$$Spread = \max(\text{detune}) - \min(\text{detune})$$

Это создает эффект хоруса и делает звук "жирным" без использования дополнительных эффектов.

---

### 3.2 Огибающие (Envelopes)

**Строки:** 26-31

Пэды отличаются очень медленным развитием звука:
- **Attack (2 сек):** Плавное нарастание громкости при нажатии.
- **Release (3 сек):** Звук продолжает звучать еще 3 секунды после того, как нота была отпущена, создавая длинный "хвост".

---

### 3.3 Математика яркости (Brightness)

**Строки:** 42-44

Параметр яркости мапится на частоту лоу-пасс фильтра в диапазоне от 200 Гц до 4200 Гц.

**Формула:**
$$Freq_{cutoff} = Brightness \times 4000 + 200$$

Используется метод `exponentialRampTo`, который обеспечивает музыкальное, логарифмическое изменение частоты, более естественное для человеческого слуха.
