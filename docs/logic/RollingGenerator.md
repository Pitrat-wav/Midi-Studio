# 📘 АУДИТ: RollingGenerator.ts

## 1. Глобальное Описание

### Назначение
Этот файл реализует **Rolling Generator** — специализированный алгоритм для создания ритмических басовых партий, характерных для электронной музыки (Trance, Techno, Psytrance). В отличие от мелодического генератора "Sting", этот фокус на ритмических фигурах: "offbeat" (слабая доля), "galloping" (галоп) и синкопированных паттернах.

### Связи
- **Импорты:** Использует интерфейс `BassStep` из `StingGenerator.ts`.
- **Используется в:** `src/store/instrumentStore.ts` для FM-синтезатора (FM Bass).
- **Связь с UI:** Выбор режима (Rolling Mode) в интерфейсе баса.

---

## 2. Интерфейс и UI Элементы

Генератор поддерживает 4 классических режима:
- **Offbeat:** Стандартный бас на "и" (вторая и четвертая 16-е доли).
- **Galloping:** Ритм "галопа" (три удара подряд, пропуск на сильной доле).
- **Syncopated:** Сложные смещения ритма, зависящие от параметра Плотности (Density).
- **Random:** Полностью случайное распределение ударов.

## 3. Логика и Математика

### 3.1 Фиксированные паттерны

**Строки:** 24-38

Алгоритм использует модульную арифметику для заполнения 16-шагового массива.

**Offbeat (Слабая доля):**
События происходят на шагах 2, 6, 10, 14.
$$Active_i = \text{true if } i \pmod 4 = 2$$

**Galloping (Галоп):**
Создает эффект "тыгыдык-тыгыдык". Бас играет везде, кроме сильных долей (где обычно звучит бочка).
$$Active_i = \text{true if } i \pmod 4 \neq 0$$

---

### 3.2 Алгоритмические паттерны

**Строки:** 40-58

**Syncopated (Синкопированный):**
Использует параметр $Density$ для вероятностного распределения, но с подавлением сильных долей, чтобы не мешать бочке.

**Математическая модель вероятности:**
$$P_i = \begin{cases} 
Density \times 0.2 & \text{если } i \pmod 4 = 0 \text{ (сильная доля)} \\
Density & \text{иначе}
\end{cases}$$

Это гарантирует, что на сильной доле бас будет появляться в 5 раз реже, чем на остальных, сохраняя чистоту микса.

**Random (Случайный):**
$$Active_i = \text{true if } \text{random}() < Density$$
