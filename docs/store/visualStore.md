# visualStore.ts

### 1. Глобальное Описание (File Overview)
* **Назначение:** Глобальное состояние визуальной системы приложения. Управляет пресетами окружения, реестром визуализаторов, энергией аудио-реактивности и данными трекинга (Hand/Pose).
* **Связи:** 
    * Используется: Почти во всех WebGL компонентах и HUD экранах.
    * Содержит: Полный список доступных визуализаторов (165 шт).

### 2. Интерфейс и Элементы Системы (System Elements)

* **Реестр: `PRESETS`**
    * **Где находится:** Строки 3-16.
    * **Логика:** Список настроек звездного неба, туманностей и освещения (например, "ORION NEBULA", "SOUTH PARK ROCK").

* **Реестр: `VISUALIZER_REGISTRY`**
    * **Где находится:** Строки 18-195.
    * **Логика:** База данных всех визуализаторов (ID 0-165) с иконками и тегами для поиска.

* **Состояние: `showTerminal` и `terminalHistory`**
    * **Где находится:** Строки 425-428.
    * **Логика:** Управляет видимостью Live Coding терминала и хранит лог команд/ответов.

### 3. Логика и Математика (Logic & Math)

* **Функция:** `decay()`
* **Что делает:** Постепенно уменьшает значения энергии и импульсов в каждом кадре анимации для создания эффекта плавного затухания.
* **Математическая модель (Decay):**
    $$ V_{next} = \max(0, V_{current} - Rate) $$
    Пример скоростей затухания:
    * Инструменты (drums, bass): 0.05
    * Импульсы (kick, snare): 0.15 - 0.2
    * Интенсивность аудио: 0.1
    * Модификаторы визуализации: $V \times 0.95$ (экспоненциальное затухание).

* **Функция:** `cycleVisualizer(dir)`
* **Что делает:** Переключает визуализаторы по кругу.
* **Математическая модель:**
    $$ Index_{next} = (Index + dir + N) \pmod N $$
    Где $N$ — общее количество визуализаторов в реестре.

* **Функция:** `setStatus(msg)`
* **Логика:** Устанавливает сообщение на 2000 мс (использует `setTimeout`), после чего сбрасывает его в `null`.

* **Построчный разбор (`updateEnergy`):**
    1. Принимает имя инструмента и новое значение (0..1).
    2. Применяет "иммутабельное" обновление объекта `energy` через spread-оператор.
