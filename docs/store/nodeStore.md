# 📘 АУДИТ: nodeStore.ts

## 1. Глобальное Описание

### Назначение
`nodeStore` — это основное хранилище состояния для модульного редактора **The Grid**, построенное на базе `Zustand`. Оно управляет списком узлов, связями между ними и их внутренними параметрами. Это "мозг" системы, который связывает визуальный интерфейс ReactFlow с аудио-процессором GraphEngine.

### Ключевые структуры
- **NODE_DEFS:** Манифест всех доступных типов узлов с их портами ввода/вывода, категориями и дефолтными параметрами.
- **NodeType:** Строго типизированный список всех идентификаторов модулей.

---

## 2. Логика хранилища

### addNode
Метод создания нового узла. При создании автоматически внедряет "стандартные параметры" DSP:
- `inputGain`: 1.0
- `outputGain`: 1.0
- `bypass`: false
Это гарантирует предсказуемое поведение всех аудио-процессоров.
