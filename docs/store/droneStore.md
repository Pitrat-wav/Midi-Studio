# droneStore.ts

### 1. Глобальное Описание (File Overview)
* **Назначение:** Хранилище состояния Дрон-движка (Drone Engine), специализирующегося на создании бесконечных атмосферных звуков и сложных текстур. Управляет параметрами хаоса, FM-синтеза и тембральной сложности.
* **Связи:** 
    * Используется: В `DroneEngine.ts` для управления аудио-узлами и в `DroneScreen.tsx` для UI.

### 2. Интерфейс и Параметры (System Parameters)

* **Параметр: `Intensity` (The Big Knob)**
    * **Логика:** Главный регулятор (0-1), влияющий на общую насыщенность звука.

* **Параметры v2.0:**
    * `chaos`: Скорость и вероятность случайных изменений (Bernoulli/S&H).
    * `fmDepth`: Глубина негармонического металлического характера.
    * `grit`: Уровень шума и экстремальной дисторсии.
    * `nervousness`: Скорость микро-детюнинга (дрожание высоты тона).

### 3. Логика и Математика (Logic & Math)

* **Функция:** `setParam(params)`
* **Что делает:** Массово обновляет любые параметры стора через частичный объект (`Partial`).

* **Математическая модель (Parameter Mapping):**
    Хотя формулы синтеза находятся в аудио-движке, стор хранит нормализованные коэффициенты ($0 \dots 1$).
    $$ AudioParam = BaseValue \times (1.0 + ConfigValue \times Range) $$
    Где $ConfigValue$ берется из `droneStore`.

* **Построчный разбор:**
    1. Инициализирует состояние по умолчанию: `enabled: false`, `intensity: 0.5`.
    2. Предоставляет методы-сеттеры для каждого параметра.
    3. Метод `toggle` инвертирует текущее состояние `enabled`.
