# aiStore.ts

### 1. Глобальное Описание (File Overview)
* **Назначение:** Модуль интеграции с внешними AI-сервисами (Hugging Face, Replicate) для генерации текстур и визуального контента на основе текстовых промптов.
* **Связи:** 
    * Импортирует: `replicate`, `@huggingface/inference`.
    * Используется в: `AIPanel.tsx`.

### 2. Интерфейс и API (Core API)

* **Метод: `setKeys(replicate, hf)`**
    * **Логика:** Устанавливает API-ключи для доступа к сервисам. Ключи хранятся в памяти приложения.

* **Метод: `generateTexture(prompt, provider)`**
    * **Логика:** Асинхронно отправляет запрос выбранному провайдеру (по умолчанию Hugging Face).
    * **Результат:** Возвращает Blob-URL сгенерированного изображения.

### 3. Логика и Математика (Logic & Math)

* **Функция:** `generateTexture`
* **Что делает:** Управляет жизненным циклом запроса к нейросети.
* **Алгоритм:**
    1. Устанавливает флаг `isGenerating = true`.
    2. **Hugging Face (HF) Path:**
        * Использует модель `stabilityai/stable-diffusion-xl-base-1.0`.
        * Добавляет стандартный negative prompt: "blurry, low quality, distorted".
        * Преобразует полученный Blob в URL через `URL.createObjectURL(blob)`.
    3. **Replicate Path:**
        * Использует модель `sdxl`.
        * Получает массив строк (URL-адресов).
    4. **Завершение:** Добавляет URL в начало массива `generatedTextures` и сбрасывает флаг загрузки.

* **Математическая модель (Prompt Logic):**
    Запрос формируется как конкатенация пользовательского ввода и технических параметров модели.
    $$ FinalPrompt = UserInput + NegativePromptFilters $$

* **Построчный разбор (Error Handling):**
    1. Проверяет наличие ключа перед запросом. Если ключа нет — выбрасывает `Error`.
    2. В случае сбоя сети или ошибки API — записывает сообщение в `lastError`.
