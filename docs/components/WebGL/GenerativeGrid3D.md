# 📘 АУДИТ: GenerativeGrid3D.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует 3D-представление шагового секвенсора. Он превращает плоский массив логических значений (паттерн) в интерактивный ряд геометрических объектов (StepCubes), расположенных на орбите инструмента. Пользователь может видеть "бегущий огонек" (текущий шаг) и переключать состояние шагов кликом по 3D-объектам.

### Связи
- **Sequencer Loop:** Получает `currentStep` для визуализации прогресса воспроизведения.
- **Instrument Stores:** Получает массив `pattern` и вызывает callback `onToggleStep` для изменения состояния.
- **WebGL Scene:** Группируется внутри 3D моделей инструментов.

## 2. Интерфейс и UI Элементы

### StepCube (Шаг секвенсора)
**Строки:** 63-92
Базовый визуальный элемент секвенсора:
- **Geometry:** Компактный куб `boxGeometry` ([0.2, 0.2, 0.1]).
- **Active State:** Если шаг активен, он окрашивается в цвет инструмента и начинает светиться (`emissive`).
- **Scanning Beam:** Дополнительный белый луч (`planeGeometry`), который "сканирует" сетку, указывая точную позицию воспроизведения.

## 3. Логика и Математика

### 3.1 Защита данных (Safe Pattern)

**Строки:** 19-30

Для предотвращения ошибок отрисовки при быстрой смене паттернов используется мемоизация и копирование массива. Это гарантирует, что цикл рендеринга работает со стабильным снимком данных (frozen state), исключая race-conditions между аудио-движком и WebGL.

### 3.2 Динамический масштаб (Target Scale)

**Строки:** 66-71

Размер каждого шага рассчитывается динамически для обеспечения визуальной обратной связи:
**Математика масштабирования:**
$$Scale = \text{lerp}(Scale_{current}, Scale_{target}, 0.2)$$
Где $Scale_{target}$ определяется состоянием:
- **Current Step (Считывание):** $1.5$ (самый большой, эффект пульсации).
- **Active (Включено):** $1.1$ (чуть больше стандартного).
- **Inactive (Выключено):** $0.8$ (уменьшенный).

### 3.3 Линейное позиционирование

**Строки:** 49

Шаги выстраиваются в ряд по горизонтали:
$$X_i = (i \times 0.25) - 2 \text{ units}$$
Это создает центрированную 16-шаговую сетку шириной 4 единицы WebGL.
