# 📘 АУДИТ: ML1853D.tsx

## 1. Глобальное Описание

### Назначение
Компонент `ML1853D` — это 3D-визуализация классического модульного секвенсора ML-185. Его ключевая особенность — круговое расположение шагов, что позволяет наглядно видеть цикличность музыкальной фразы. Каждый шаг представлен как физический переключатель с индикацией высоты тона (Pitch) и текущего состояния воспроизведения.

### Связи
- **Sequencer Store:** Источник данных о массиве шагов (`stages`), текущем активном индексе (`currentStageIndex`) и параметрах каждого этапа.
- **Spatial Layout:** Определяет положение секвенсора в 3D сцене.

## 2. Интерфейс и UI Элементы

### Шаги секвенсора (Circular Steps)
**Строки: 17-38**
- **Geometry:** 8 шестиугольных цилиндров (`cylinderGeometry` с 6 сегментами), имитирующих ручки модульных синтезаторов.
- **Индикация активности:** Активный в данный момент шаг загорается ярко-оранжевым цветом (`#ffcc33`), остальные остаются темно-серыми.
- **Pitch Text:** Над каждым шагом отображается числовое значение высоты тона, установленное для этой стадии.

## 3. Логика и Математика

### 3.1 Круговое позиционирование

**Строки: 18-21**

Шаги распределяются по кругу радиусом 2 единицы:
$$X = \cos(\frac{index}{8} \times 2\pi) \times 2$$
$$Z = \sin(\frac{index}{8} \times 2\pi) \times 2$$

### 3.2 Автоматическая ориентация

**Строки: 25**

Каждый шаг развернут лицом к центру или от центра для сохранения симметрии:
$$Rotation.y = -\frac{index}{8} \times 2\pi$$
