# 📘 АУДИТ: Sampler3D.tsx

## 1. Глобальное Описание

### Назначение
Компонент `Sampler3D` — это высокодетализированная 3D-модель легендарной рабочей станции Fairlight CMI IIx, интегрированная в виртуальное пространство студии. Он служит визуальным и интерактивным продолжением 2D-интерфейса `CMISamplerView`. С помощью этого модуля пользователь может видеть работу секвенсора Page R, наблюдать спектральную осциллограмму на ЭЛТ-мониторе и переключать семплы прямо в 3D.

### Связи
- **Sampler Store:** Управляет состоянием сетки (Grid), выбором файлов и гранулярными параметрами.
- **AudioVisual Bridge:** Передает FFT-данные для формирования "векторной" волны на экране.
- **CRT Shader:** Специализированный шейдер, создающий эффект старого монитора с разверткой и шумами.

## 2. Интерфейс и UI Элементы

### ЭЛТ-Монитор (CRT Screen)
**Строки: 118-150**
- **Shader FX:** Использование `CRTShader` для имитации выпуклого стекла, строк развертки и характерного зеленоватого свечения.
- **PageRGrid:** Упрощенная 3D-версия сетки 8x16, где клетки загораются в такт ритму.
- **VectorWaveform:** Динамическая линия, отрисовывающая спектр аудиосигнала.

### Корпус и Оборудование (Hardware)
**Строки: 105-177**
- **Main Chassis:** Большой серый корпус ("холодильник"), характерный для оригинального оборудования.
- **Alpha Keyboard:** Отдельный блок клавиатуры с проволочной сеткой (wireframe) для обозначения клавиш.
- **Navigation Buttons:** Физические красная и зеленая кнопки для переключения семплов в библиотеке.

## 3. Логика и Математика

### 3.1 Векторная отрисовка спектра (Vector Line)

**Строки: 42-82**

В отличие от 2D Canvas, здесь используется `THREE.Line` с 64 вершинами. Координата Y каждой вершины обновляется в каждом кадре:
$$Y_i = \frac{FFTData_i}{255} \times 0.4$$
Геометрия помечается флагом `needsUpdate = true`, что позволяет эффективно изменять форму линии без пересоздания объекта.

### 3.2 Синхронизация 3D-сетки (Page R)

**Строки: 92-96**

Активный шаг вычисляется на основе времени исполнения (`state.clock.elapsedTime`) и BPM, обеспечивая идеальное совпадение визуального курсора на мониторе с аудио-событиями:
$$Step = \lfloor \frac{Time}{\text{StepTime}} \rfloor \pmod{16}$$

### 3.3 Интерактивность в пространстве

Компонент использует `e.stopPropagation()` для всех кликов по экранным ячейкам или кнопкам навигации. Это гарантирует, что при взаимодействии с семплером камера не будет пытаться сфокусироваться на других объектах.
