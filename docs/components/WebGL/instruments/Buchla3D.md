# 📘 АУДИТ: Buchla3D.tsx

## 1. Глобальное Описание

### Назначение
Компонент `Buchla3D` представляет собой внешнюю оболочку (рековую панель) для сложного генератора волноформ Buchla 259. Этот модуль служит точкой входа в интерфейс управления синтезатором вест-кост стиля. Модель имитирует классическую эстетику Buchla с характерными синими ручками и алюминиевой передней панелью. При клике на объект камера фокусируется на инструменте, открывая 2D-интерфейс `Buchla259Screen`.

### Связи
- **Spatial Layout:** Позиционирует модуль в стойке синтезаторов.
- **Visual Store:** Отслеживает фокус (`focusInstrument === 'buchla'`) для активации световой индикации на корпусе.

---

## 2. Интерфейс и UI Элементы

### Корпус и Панель
**Строки: 17-31**
- **Main Body:** Алюминиевый бокс с металлическим блеском (`metalness: 0.8`).
- **Focus Glow:** При выборе инструмента корпус начинает мягко светиться синим цветом, указывая на его активность.
- **Graphic Overlays:** Тонкие линии на передней панели, имитирующие разметку оригинального модуля.

### Органы управления (Decorative Knobs)
**Строки: 34-46**
- **Blue Knobs:** Фирменные синие регуляторы Buchla, являющиеся узнаваемым элементом дизайна.

---

## 3. Логика и Математика

### 3.1 Фокусировка и взаимодействие

**Строки: 11-13, 17**

Компонент реализует метод "Click-to-Focus". При нажатии на любую часть корпуса вызывается `setFocus('buchla')`, что инициирует плавное перемещение камеры через `CameraController` и переключение HUD-интерфейса.
