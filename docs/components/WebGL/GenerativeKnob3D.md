# 📘 АУДИТ: GenerativeKnob3D.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует 3D-регулятор (Knob) для управления аудио-параметрами непосредственно в пространстве. Он позволяет пользователю настраивать частоту, резонанс и другие значения инструментов, взаимодействуя с ними мышью или касанием. Визуально ручка напоминает высокотехнологичное оборудование с неоновой индикацией и динамическим "гало" эффектом.

### Связи
- **Visual Store:** Передает состояние `interactionEnergy` и устанавливает `activeParam` при захвате ручки. Изменят переменную `setInteraction` для управления глобальной реакцией сцены.
- **R3F Scene:** Используется во всех 3D-панелях инструментов.

## 2. Интерфейс и UI Элементы

### Дизайн контроллера
- **Knob Body:** Цилиндрический корпус (`cylinderGeometry`) с металлической текстурой. Светится при взаимодействии (`emissive`).
- **Indicator:** Физический маркер на поверхности ручки, показывающий текущее положение.
- **Halo Ring:** Световое кольцо (`ringGeometry`) под ручкой, которое становится ярче, когда пользователь начинает регулировку.

## 3. Логика и Математика

### 3.1 Физическое вращение

**Строки:** 23-28

Позиция ручки рассчитывается на основе нормализованного значения параметра:
$$Normalized = \frac{Value - Min}{Max - Min}$$
$$Rotation_{target} = (Normalized \times 270^{\circ}) - 135^{\circ}$$
Для плавности движения используется интерполяция `lerp` с коэффициентом 0.1.

### 3.2 Механика управления (Vertical Drag)

**Строки:** 47-54

Для настройки используется вертикальное перемещение:
$$Delta = Movement_Y \times -0.01$$
$$Value_{new} = \text{clamp}(Value_{old} + Delta \times (Max - Min), Min, Max)$$
Инверсия (`-0.01`) позволяет увеличивать параметр движением мыши "вверх".

### 3.3 Визуальный отклик (Feedback)

**Строки:** 29-30, 65

Когда ручка находится в режиме редактирования (`isEditing`):
1. **Масштаб:** Ручка увеличивается на 20% в зависимости от энергии взаимодействия.
2. **Свечение:** Интенсивность `emissive` возрастает в 10 раз, создавая акцент на активном параметре.
