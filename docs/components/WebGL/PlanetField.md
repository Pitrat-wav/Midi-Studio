# 📘 АУДИТ: PlanetField.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент наполняет 3D-сцену декоративными объектами (планетами), создавая ощущение глубины и масштабности. Он поддерживает два визуальных режима: "Cosmic" (реалистичные блестящие сферы) и "Cartoon" (плоские стилизованные объекты). Особенностью компонента является прямая интеграция с AI-движком: если пользователь генерирует новую текстуру через `AIPanel`, она мгновенно накладывается на все планеты в поле видимости.

### Связи
- **AI Store:** Получает `currentTextureUrl` для динамического обновления материалов.
- **Three.js / Drei:** Использует `MeshDistortMaterial` для органических деформаций и `useTexture` для загрузки изображений.
- **Generative Background:** Является важным подмодулем общего фона студии.

## 2. Интерфейс и UI Элементы

### Типы планет
1. **Cosmic (Космические):** Металлические сферы высокой детализации с эффектом искажения (distort). Они отражают свет и выглядят как технологичные обломки или далекие звезды.
2. **Cartoon (Мультипликационные):** Стилизованные сферы с плоским затенением (`flatShading`). Используются в теме South Park.

## 3. Логика и Математика

### 3.1 Процедурная генерация поля (Planet Distribution)

**Строки:** 20-42

При инициализации создается массив планет со случайными параметрами:
- **Angle:** Расположение по кругу $(\frac{i}{N} \times 2\pi)$.
- **Distance:** Удаление от центра от 30 до 70 единиц.
- **Scale:** Случайный масштаб от 1x до 4x.
- **Speed & Orbit:** Индивидуальные параметры вращения и смещения.

### 3.2 Орбитальная анимация и Квантование (Animation)

**Строки:** 62-79

В каждом кадре позиция планеты обновляется по круговой траектории:
$$X = X_{base} + \sin(Time \times Speed + Orbit) \times 2$$
$$Y = Y_{base} + \cos(Time \times Speed + Orbit) \times 2$$

**Эффект "South Park" (Stop-motion):**
Для придания мультяшного стиля в режиме `cartoon` используется пространственное квантование (ступенчатое движение):
$$Pos_{quantized} = \frac{\text{floor}(Pos \times 12)}{12}$$
Это создает эффект "дерганой" анимации, характерной для классической перекладочной анимации.

### 3.3 Интеграция с ИИ (AI Texture Mapping)

**Строки:** 56-60, 88, 93-100

Если в `aiStore` появляется ссылка на новую текстуру, хук `useTexture` загружает её, и она автоматически подменяет основной цвет материала или смешивается с ним. Для режима Cosmic при наличии текстуры снижается металличность (`metalness`) и увеличивается шероховатость (`roughness`), чтобы детали текстуры были лучше видны.
