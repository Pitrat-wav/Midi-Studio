# 📘 АУДИТ: DrumPad3D.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует интерактивный 3D-пэд для управления ударными инструментами. Каждый пэд физически представлен в 3D сцене как "умный куб", который пульсирует, светится и деформируется в реальном времени при воспроизведении соответствующего звука (Kick, Snare и т.д.) или при нажатии пользователем. Это ключевой элемент тактильного взаимодействия в 3D студии.

### Связи
- **AudioVisual Bridge:** Получает высокоточные триггеры пульсации для конкретного `id` инструмента.
- **Instrument Selector:** Взаимодействует с логикой выбора инструментов.
- **Visuals:** Использует сложные шейдерные материалы для визуализации "энергии" звука.

---

## 2. Интерфейс и UI Элементы

### Визуальное воплощение
- **Geometry:** Прямоугольный блок `boxGeometry` ([0.8, 0.8, 0.2]), имитирующий пэды классических MIDI-контроллеров (например, MPC).
- **Material:** `MeshDistortMaterial`. Материал обладает металлическим блеском (`metalness: 0.8`) и способностью искажать свою форму при получении аудио-импульса.
- **Emissive Lighting:** Пэд излучает свет того же цвета, что и его основной материал, причем интенсивность свечения напрямую зависит от силы звукового удара.

## 3. Логика и Математика

### 3.1 Физическая симуляция пульсации

**Строки:** 33-35, 41-45

При получении сигнала `trigger` от аудио-движка, пэд меняет свои физические свойства:
1. **Масштаб:** Резкое увеличение размера с последующим плавным затуханием (Lerp).
   $$Scale_{target} = 1.0 + (Trigger \times 0.4)$$
2. **Искажение (Distort):** Повышение коэффициента деформации геометрии.
   $$Distort = 0.2 + (Trigger \times 0.5)$$
3. **Скорость анимации материала:** Ускорение внутренних "волн" материала.
   $$Speed = 1 + (Trigger \times 10)$$

### 3.2 Эффект левитации (Floating)

**Строки:** 38

Для придания сцене "органичности", каждый пэд плавно покачивается в воздухе по синусоиде. Смещение фазы синуса зависит от X-координаты пэда, что предотвращает синхронное движение всех пэдов.
$$Y = Y_{base} + \sin(Time \times 2 + X_{pos}) \times 0.05$$
