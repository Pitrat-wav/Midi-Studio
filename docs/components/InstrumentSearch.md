# 📘 АУДИТ: InstrumentSearch.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует "командную палитру" (Command Palette) или быстрый поиск (Spotlight) по инструментам студии. Он позволяет пользователю мгновенно перемещаться между модулями, вводя их название, вместо того чтобы искать их в 3D пространстве или кликать по иконкам. Это продвинутый инструмент навигации для опытных пользователей (Power Users).

### Связи
- **Spatial Layout:** Получает список доступных ключей инструментов из `SPATIAL_LAYOUT`.
- **Navigation:** Вызывает функцию `onSelect`, которая обычно обновляет фокус камеры в `visualStore`.
- **Shortcuts:** Интегрирован с глобальной клавиатурой (Cmd+K).

---

## 2. Интерфейс и UI Элементы

### Модальное окно поиска
- **Input Field:** Текстовое поле с авто-фокусом для ввода поискового запроса.
- **Results List:** Динамический список совпадений. Активный элемент подсвечивается.
- **Human-readable Names:** Маппинг `INSTRUMENT_NAMES` преобразует технические ключи (например, `buchla`) в понятные названия ("Buchla 259 Complex Generator").
- **Kbd Hints:** Визуальные подсказки горячих клавиш (ESC, Enter, стрелки).

## 3. Логика и Математика

### 3.1 Алгоритм фильтрации

**Строки:** 32-37

Поиск выполняется регистронезависимо по двум полям: техническому ключу инструмента и его человекочитаемому названию.
$$Result = Keys.filter(k \rightarrow Name_k.includes(Query) \lor k.includes(Query))$$

### 3.2 Управление фокусом и навигацией

**Строки:** 41-77

Компонент реализует собственную логику обработки клавиш внутри `useEffect`:
- **Открытие:** `Cmd+K` или `/`.
- **Циклическая навигация:** Стрелки `Up/Down` меняют `selectedIndex` с использованием оператора остатка от деления:
  $Index_{new} = (Index_{old} + 1) \pmod{Results.length}$
- **Выбор:** `Enter` подтверждает выбор подсвеченного инструмента.
