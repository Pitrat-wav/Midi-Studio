# 📘 АУДИТ: FAQ.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент представляет собой информационное окно помощи (Knowledge Base), которое знакомит пользователя с управлением приложением. Оно содержит структурированный список горячих клавиш, описание навигации в 3D пространстве и полезные советы по использованию продвинутых функций (Camera Fly Mode, Hand Tracking, Vocoder).

### Связи
- **Navigation:** Вызывается из главного меню или по нажатию клавиши `?`.
- **Styling:** Использует CSS-класс `.glass` для создания эффекта матового стекла, соответствующего общей эстетике приложения.

---

## 2. Интерфейс и UI Элементы

### Структура разделов
1. **Transport:** Команды управления воспроизведением (Space) и системные вызовы (H, ?, P).
2. **Navigation:** Схема управления камерой в 3D сцене (WASD, QE) и быстрые переходы по цифровым клавишам (0-9).
3. **Controls:** Команды управления звуком выбранного инструмента (Mute, Volume).
4. **Tips (Советы):** Инструкции по активации микрофона, видео-трекинга и работе с 3D-манипуляторами.

### Визуальные акценты
- Использование тегов `<kbd>` для стилизации клавиш клавиатуры.
- Текстовая легенда для расшифровки цифровых переходов (1: Drums ... 9: Master).
- Акцент на кнопке **PANIC** (клавиша P) как на критически важном элементе управления.

---

## 3. Логика и Математика

Компонент является статическим (презентационным) и не содержит сложной вычислительной логики. Его основная задача — визуализировать маппинг, определенный в `KeyboardController.tsx`, обеспечивая синхронизацию документации с реальным функционалом приложения.
