# 📘 АУДИТ: DroneScreen.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент является панелью управления **Infinite Drone Engine** — специализированным генератором бесконечных звуковых ландшафтов. Вдохновленный эстетикой дрон-музыкального минимализма, он позволяет активировать мощный синтезированный слой из гармонических осцилляций, который служит фундаментом для всей композиции.

### Связи
- **Store:** Управляет флагом `isDroneEnabled` в `useHarmStore`.
- **Sound Design:** Активирует фоновый синтез, не зависящий от ритма секвенсора.
- **Visuals:** Использует `visualStore` для управления фокусом камеры.

## 2. Интерфейс и UI Элементы

### Drone Orb (Силовая ячейка)
**Строки:** 21-32
Центральный интерактивный элемент ("орб"), выполняющий роль стилизованного выключателя питания. При активации внутреннее ядро начинает светиться, а статус меняется на "DRONE ACTIVE".

### Visual Feedback (Кольца резонанса)
**Строки:** 35-52
Уникальная визуализация, состоящая из 30 концентрических колец.
- **Пульсация:** В активном режиме кольца анимируются с разной скоростью (`pulse-ring`), создавая эффект интерференции волн.
- **Градиент прозрачности:** Кольца затухают по мере удаления от центра, визуализируя распространение звука в пространстве.

## 3. Логика и Математика

### 3.1 Генерация концентрических колец

**Строки:** 36-39

Геометрия колец рассчитывается динамически в цикле:
**Математика размеров:**
$$Radius_i = 40 + (i \times 3) \text{ px}$$
$$Opacity_i = \text{isActive} ? (0.8 - i \times 0.025) : 0.1$$

Каждое кольцо получает индивидуальное время анимации:
$$Duration_i = 4 + i \times 0.2 \text{ s}$$
Это создает эффект "органического" движения, так как фазы пульсации колец не совпадают, что математически имитирует сложные биения (beating), характерные для дрон-музыки.
