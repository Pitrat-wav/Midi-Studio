# 📘 АУДИТ: NodeInspector.tsx

## 1. Глобальное Описание

### Назначение
Компонент `NodeInspector` является профессиональной панелью управления параметрами конкретного узла в **The Grid**. Он реализует интерфейс в стиле "TouchDesigner", группируя параметры по функциональным секциям (Input, Core, Modulation, Output) и обеспечивая глубокий контроль над DSP-процессом.

### Связи
- **Node Store:** Читает состояние узла и обновляет параметры через `updateNodeParam`.
- **NodeEditor:** Вызывается при двойном клике по узлу.
- **GraphEngine:** Получает изменения параметров в реальном времени для обновления аудио-движка.

---

## 2. Интерфейс и UI Элементы

### Секции параметров
1.  **INPUT PROCESSING:** Входной гейн и предварительная фильтрация (если применимо).
2.  **MODULE PARAMETERS:** Основные параметры узла (частота, тип волны, добротность и т.д.).
3.  **MODULATION:** Настройки LFO, огибающих и других источников модуляции.
4.  **OUTPUT & GAIN:** Финальный гейн, Mix (Wet/Dry), Pan и кнопка Bypass.

### Визуальные элементы
- **Signal Chain Preview:** Визуальное отображение прохождения сигнала через узел с пульсирующей анимацией активности.
- **Modern Sliders:** Кастомные слайдеры с индикацией текущего значения.
- **Status Dot:** Индикатор активности узла.
