# 📘 АУДИТ: Buchla259Screen.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент представляет собой детальную реплику интерфейса легендарного синтезаторного модуля **Buchla 259 "Complex Waveform Generator"**. Он переносит философию "West Coast Synthesis" в цифровое пространство, позволяя пользователю управлять сложными методами тембрального формирования (wavefolding, FM/AM модуляция) через аутентичный визуальный интерфейс.

### Связи
- **Store:** Управляет параметрами в `useHarmStore` (настройки `HarmSynth`).
- **Synthesis:** Напрямую влияет на алгоритмы формирования волны в `HarmSynth.ts`.
- **Navigation:** Вызывается при фокусировке на модуле Buchla в 3D пространстве.

## 2. Интерфейс и UI Элементы

### Секция Modulation Oscillator (Левая колонка)
Источник модулирующего сигнала:
- **Frequency:** Большая синяя ручка для настройки частоты модуляции.
- **Waveshape:** Переключатели (LED) для выбора формы волны (Sine, Saw, Square).

### Секция Control (Центральная колонка)
"Сердце" патч-панели:
- **Mod Toggles:** Переключатели назначения модуляции (на Pitch, Amp или Timbre).
- **Mod Index:** Глубина влияния модулирующего осциллятора на основной.

### Секция Principal Oscillator (Правая колонка)
Основной звуковой источник:
- **Pitch:** Основная частота звука.
- **Shaping Grid:** Тонкая настройка тембра (Timbre — степень сворачивания волны, Harmonics — спектральная плотность, Order — расстановка гармоник).

## 3. Логика и Математика

### 3.1 Модель Buchla 259

Экран реализует специфическую топологию Buchla, где один осциллятор (Mod) "атакует" параметры другого (Principal).
**Логика модуляции:**
При включенных тумблерах `complexPitchMod` или `complexTimbreMod`, значения ползунков в сторе интерпретируются `HarmSynth.ts` как коэффициенты для LFO или FM-синтеза.

### 3.2 Визуальная эмуляция (Hardware UI)

Использование CSS для имитации классических "Buchla Knobs" и металлических тумблеров. Каждый `input type="range"` стилизован под поворотную ручку с маркером положения. Переключение состояния (Toggle) сопровождается сменой визуального класса `.on`, который активирует свечение виртуальных LED-индикаторов.
