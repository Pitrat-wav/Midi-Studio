# 📘 АУДИТ: MacroRack.tsx

## 1. Глобальное Описание

### Назначение
`MacroRack` — это панель глобального управления (Instrument Rack), предоставляющая быстрый доступ к 8 наиболее важным параметрам текущей сборки в Node Editor. Она позволяет исполнителю управлять сложными модуляциями через простые макро-регуляторы.

### Связи
- **Store:** Активно использует `nodeStore` (массив `macros` и методы `setMacroValue`, `updateMacroLabel`).
- **Components:** Использует компонент `Knob` для отрисовки регуляторов.

---

## 2. Интерфейс и UI Элементы

- **Macro Grid:** Сетка 4x2, содержащая 8 макро-модулей.
- **Knob Wrapper:** Содержит энкодер с цветовой индикацией статуса маппинга.
- **Label Input:** Текстовое поле, позволяющее переименовать макрос прямо во время исполнения.
- **Target Info:** Маленькая метка внизу каждого макроса, отображающая имя целевого параметра (например, "FREQ" или "GAIN").

---

## 3. Логика и Математика

### 3.1 Управление состоянием
Компонент является "глухим" потребителем состояния из `nodeStore`. Все изменения (вращение ручки, ввод текста) мгновенно транслируются в стор через экшены.

### 3.2 Визуальная обратная связь
- **Mapped State:** Если у макроса есть `targetNodeId`, он подсвечивается синим цветом ($$#007aff$$).
- **Unmapped State:** Серый цвет ($$#333$$) и иконка `ZapOff`, указывающая на отсутствие связи.
