# 📘 АУДИТ: MasterScreen.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент является центральным пультом управления (Global Control Center) всей студии. Он объединяет в себе основные функции транспорта (Play/Stop), управление темпом (BPM) и мастер-микшер для всех аудио-каналов. Это место, где пользователь принимает глобальные решения по динамике и структуре звучания всего проекта.

### Связи
- **Audio Store:** Главный потребитель данных из `audioStore` (BPM, Volumes, Mutes, Transport State).
- **Tone.js:** Проксирует команды управления темпом и воспроизведением напрямую в аудио-движок.
- **Visuals:** Позволяет закрывать HUD через `visualStore`.

## 2. Интерфейс и UI Элементы

### Transport Controls (Транспорт)
**Строки:** 21-34
- **Play/Pause:** Переключение состояния воспроизведения всей студии.
- **PANIC:** Критически важная кнопка для мгновенной остановки всех звуков и сброса аудио-контекста (экстренная помощь при "зависших" нотах).

### BPM Section (Темп)
**Строки:** 37-56
- **Display:** Крупное табло с текущим значением ударов в минуту.
- **Slider:** Ползунок для плавной регулировки темпа (от 60 до 200 BPM).
- **Presets:** Кнопки быстрого перехода к стандартным значениям темла (90 — HipHop, 120 — House, 140 — Dubstep, 174 — DnB).

### Channel Mixer (Микшер каналов)
**Строки:** 59-80
Вертикальная сетка индикаторов для каждого инструмента (Drums, Bass, Lead и т.д.):
- **Meter Fill:** Визуализация текущего уровня громкости (от 0 до 100%).
- **Mute Кнопка:** Кнопка быстрого отключения звука выбранного канала.

## 3. Логика и Математика

### 3.1 Глобальный темп

**Строки:** 42-49

Изменение BPM в сторе вызывает мгновенное обновление частоты тиков в `Tone.Transport`, что синхронно ускоряет или замедляет все запущенные секвенсоры и эффекты задержки (delay).

### 3.2 Маппинг уровней (Volume Mapping)

**Строки:** 60-71

Массив громкостей из стора итерируется для построения микшера. Высота индикатора уровня рассчитывается линейно:
$$Height = Volume_{0..1} \times 100\%$$
Это обеспечивает наглядную индикацию энергетического баланса сессии.
