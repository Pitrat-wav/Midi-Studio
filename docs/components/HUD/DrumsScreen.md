# 📘 АУДИТ: DrumsScreen.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент представляет собой детальную реплику классических драм-машин (TR-808 / TR-909) в формате 2D HUD. Он служит основным интерфейсом для программирования ударных, изменения звуковых наборов (kits) и точной настройки синтеза каждого барабана (Kick, Snare, Hi-Hat и др.). Интерфейс сочетает в себе винтажную эстетику аналогового оборудования с современными алгоритмами генерации ритмов.

### Связи
- **Drum Store:** Управляет состоянием всех инструментов ударной установки, их параметрами и паттернами.
- **Visual Store:** Обеспечивает выход из режима фокусировки (`setFocusInstrument(null)`).
- **CSS Themes:** Динамически меняет оформление всей панели в зависимости от выбранного кита (`808` или `909`).

## 2. Интерфейс и UI Элементы

### Структура панели (Hardware Panel)
**Строки: 29-53**
- **Branding:** Динамические заголовки, имитирующие реальное оборудование Roland.
- **Master Controls:** Глобальные кнопки START/STOP, выбор кита и кнопка выключения питания (OFF).
- **Side Panels:** Декоративные деревянные или пластиковые боковины, усиливающие эффект физического устройства.

### Секции инструментов (Inst-Sections)
**Строки: 55-95**
Каждый из 6 типов барабанов имеет свою секцию настроек:
- **Pitch:** Регулировка тональности удара.
- **Decay:** Длительность хвоста звука.
- **Level:** Индивидуальная громкость (Gain) в децибелах.

### Редактор эвклидовых паттернов
**Строки: 97-128**
- **Steps/Pulses:** Позволяет задавать параметры алгоритма Бьёрклунда для автоматической генерации "умных" ритмов.
- **Hardware Steps:** Ряд из 16 физических кнопок с LED-индикаторами. Каждая кнопка позволяет вручную включать/выключать удары.

## 3. Логика и Математика

### 3.1 Динамическая типизация параметров

**Строки: 18-20, 57**

Поскольку состояние стора содержит объекты для каждого барабана, компонент использует строковые ключи для доступа к данным:
`const drumState = (store as any)[d.id]`
Это позволяет использовать единый обработчик `handleParamChange` для всех типов регуляторов и всех инструментов.

### 3.2 Синхронизация паттернов

**Строки: 116-127**

Компонент отображает массив `activePatterns` для выбранного барабана. Каждое изменение (клик по кнопке или изменение числа импульсов) немедленно пересчитывает состояние в `drumStore`, что приводит к обновлению визуализации как в 2D, так и в 3D мире.

### 3.3 Тематическое переключение (Layout Mapping)

Изменение `store.kit` влечет за собой смену CSS-классов:
- **808:** Оранжево-красная гамма "Computer Composer".
- **909:** Серо-бежевая гамма "Digital Drum System".
