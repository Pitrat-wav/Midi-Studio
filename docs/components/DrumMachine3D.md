# 📘 АУДИТ: DrumMachine3D.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент является самой сложной и визуально насыщенной частью 3D-студии. **Generative Drum Lab** — это виртуальная драм-машина, объединяющая в себе физическую симуляцию (Kick/Snare), управление генеративными ритмами через алгоритмы Бьёрклунда и интерактивную консоль управления "Mission Control". Она превращает программирование барабанов в пространственный опыт, где звук можно буквально видеть и трогать.

### Связи
- **Drum Store:** Управляет всеми параметрами синтеза (Pitch, Decay) и алгоритмической композиции (Steps, Pulses).
- **AudioVisual Bridge:** Синхронизирует деформации 3D-моделей с каждым ударом барабана.
- **Custom Shaders:** Использует специализированные шейдеры для создания эффекта Френеля и аудио-реактивных деформаций.

## 2. Интерфейс и UI Элементы

### Визуальные инструменты (Generative Visuals)
1. **Kick Drum (Ядро):** Центральная красная сфера, защищенная "стеклянной оболочкой" (`MeshTransmissionMaterial`). Она пульсирует и меняет форму при каждом ударе.
2. **Snare Drum (Тетраэдр):** Синий сложный многогранник, совершающий резкие вращения и деформации при ударе малого барабана.
3. **HiHat Particles:** Рой из 40 золотых частиц, которые ускоряют свое вращение и разлетаются при звуке хай-хэта.

### Алгоритмические кольца (Bjorklund Rings)
**Строки: 144-205**
Три вертикально расположенных голографических кольца вокруг центрального ядра. Они визуализируют эвклидовы ритмы. Каждое кольцо разделено на сегменты (шаги), яркость которых указывает на наличие удара и текущую позицию воспроизведения.

### Консоль управления (Control Console)
**Строки: 208-276**
Наклонный пульт управления, содержащий три идентичные секции (Channel Strips) для управления параметрами каждого звука. Каждая секция включает:
- **PITCH / DECAY:** 3D-ручки для настройки тембра.
- **STEPS / PULSES:** 3D-ручки для управления алгоритмом генерации ритма.
- **TRIG:** Физическая кнопка для ручного запуска звука.

## 3. Логика и Математика

### 3.1 Алгоритм Бьёрклунда (Euclidean Rhythm)

**Строки: 153-164**

Ритмы генерируются на лету по алгоритму равномерного распределения импульсов:
**Математика:**
Для каждого шага $i \in [0, Steps]$:
$$Count = Count + Pulses$$
$$\text{If } Count \ge Steps \text{ then } \text{Trigger } = \text{true}, Count = Count - Steps$$
Это гарантирует математически идеальное распределение ударов, создавая музыкальные ритмы, характерные для этнической и авангардной музыки.

### 3.2 Аудио-реактивные шейдеры (Shaders)

**Строки: 64-68**

Для Kick Drum используется `shaderMaterial` с двумя компонентами:
1. **Vertex Shader:** Искажает геометрию сферы на основе `uAudioIntensity`.
2. **Fresnel Fragment Shader:** Создает эффект внутреннего свечения и прозрачности по краям объекта, имитируя высокотехнологичную плазму.

### 3.3 Плавная анимация (Interpolation)

Все изменения масштаба и положения ручек проходят через фильтр `lerp` с коэффициентами от 0.1 до 0.25, что придает интерфейсу "инерцию" и ощущение физической массы.
