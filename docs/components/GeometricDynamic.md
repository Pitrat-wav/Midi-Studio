## 2. Логика и Математика

### 2.1 Instanced Mesh (Оптимизация)
Вместо создания 1024 отдельных объектов, используется `InstancedMesh`. Это позволяет отрисовать все кубы за один вызов отрисовки (draw call), что критически важно для производительности.

### 2.2 Распределение (Fibonacci Sphere)
Для равномерного распределения точек по поверхности сферы используется алгоритм, основанный на золотом сечении:
$$phi = \arccos(-1 + \frac{2 \times i}{N})$$
$$theta = \sqrt{N \times \pi} \times phi$$
Это создает эстетически приятную и сбалансированную структуру.

### 2.3 Аудио-Реактивность (FFT Mapping)
Каждый куб привязан к конкретному диапазону частот спектра:
$$Scale = 1.0 + \frac{FFT[freq\_index]}{255} \times 5.0$$
$$Radius = 5 + \sin(Time + i) \times 2 \times AudioIntensity$$
