# 📘 АУДИТ: ContextMenu.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует контекстное меню (выпадающее меню по правому клику или долгому нажатию), предназначенное в первую очередь для редактора модулей (Node Editor). Оно позволяет выполнять быстрые действия над узлами графа, такие как дублирование или удаление, экономя время пользователя на поиск нужных кнопок в интерфейсе.

### Связи
- **Node Editor:** Вызывается при взаимодействии с холстом или конкретным узлом в `NodeEditor.tsx`.
- **Icons:** Использует `lucide-react` для визуального обозначения действий (Trash2, Copy, X).

---

## 2. Интерфейс и UI Элементы

### Структура меню
- **Header:** Заголовок, указывающий на объект действия (NODE ACTIONS или CANVAS), с кнопкой закрытия.
- **Duplicate Action:** Кнопка копирования выбранного узла со всеми его текущими параметрами.
- **Delete Action:** Кнопка удаления узла из графа и аудио-потока (подсвечена красным цветом `delete`).
- **Empty State:** Если меню вызвано на пустом месте холста, отображается подсказка "Select a node...".

### Позиционирование
Меню использует абсолютное позиционирование (`position: 'absolute'`) на основе координат клика $(x, y)$, что гарантирует его появление именно в точке взаимодействия.

---

## 3. Логика и Математика

Компонент оптимизирован с помощью `memo`, чтобы предотвратить лишние перерисовки при движении мыши по холсту. Событие `onClick` имеет вызов `e.stopPropagation()`, чтобы предотвратить нежелательное взаимодействие с лежащими ниже элементами графа или запуск перетаскивания при работе с меню.
