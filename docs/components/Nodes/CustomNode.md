# CustomNode.tsx

### 1. Глобальное Описание (File Overview)
* **Назначение:** Основной визуальный компонент для узлов в Node Editor (React Flow). Реализует интерфейс управления параметрами ноды, встроенный редактор JavaScript кода и осциллограф для визуальной обратной связи.
* **Связи:** 
    * Импортирует: `useNodeStore`, `GraphEngine`, `react-flow`.
    * Используется в: `NodeEditor.tsx` (как кастомный тип ноды).

### 2. Интерфейс и UI Элементы (Visual Elements)

* **Элемент:** Заголовок ноды (Node Header)
    * **Где находится:** Строки 248-260.
    * **Логика:** Отображает название и иконку категории (аудио, логика, скрипт). Кнопка `X` удаляет ноду.

* **Элемент:** Встроенный Осциллограф (Canvas Scope)
    * **Где находится:** Строки 85-87, 17-78.
    * **Логика:** Отрисовывает форму волны или анимацию в реальном времени.

* **Элемент:** Редактор кода (Script Editor)
    * **Где находится:** Строки 217-232.
    * **Логика:** Полноэкранный оверлей для написания JSFX-скриптов.

* **Элемент:** Порты ввода/вывода (Handles)
    * **Где находится:** Строки 264-294.
    * **Логика:** React Flow Handles для соединения нод. Цвета зависят от типа порта (audio, event, value).

### 3. Логика и Математика (Logic & Math)

* **Функция:** Отрисовка осциллограммы (Canvas Loop)
* **Что делает:** Визуализирует аудио-данные из `Tone.Waveform`.
* **Математическая модель (Scaling):**
    $$ X_{pixel} = i \times \text{step} $$
    $$ Y_{pixel} = 30 - (SampleValue \times 28) $$
    * 30 — центр оси Y (при высоте 60).
    * 28 — коэффициент усиления амплитуды для заполнения высоты.
    * $i$ — индекс пикселя (0..140).

* **Построчный разбор (Render Loop):**
    1. Получаем инстанс ноды из `GraphEngine`.
    2. Если нода — `visual_scope`, берем текущий буфер данных через `wave.getValue()`.
    3. Рассчитываем шаг выборки: `step = Math.ceil(bufferLength / 140)`.
    4. Отрисовываем ломаную линию через `ctx.lineTo()`.

* **Логика параметров (`renderContent`):**
    Динамически выбирает UI-контрол на основе типа данных и имени ключа:
    * `frequency` -> Range Input (20..5000 Гц).
    * `type`/`mode` -> Select (выпадающий список).
    * Массивы -> Сетка слайдеров для секвенсора.
