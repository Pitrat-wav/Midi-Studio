# 📘 АУДИТ: MasterControl3D.tsx

## 1. Глобальное Описание

### Назначение
Компонент `MasterControl3D` является "Центром Управления Полетами" (Mission Control) всей виртуальной студии. Это терминал, который объединяет глобальные параметры аудио-движка, систему пресетов, блок эффектов, эквализацию и механизмы экспорта. Он служит точкой интеграции всех отдельных инструментов, позволяя управлять композицией как единым целым.

### Связи
- **Audio Store:** Глобальный темп (BPM), мастер-громкость, свинг и FX-параметры.
- **Instrument Stores:** Прямое взаимодействие со всеми сторами (Drums, Bass, Harmony и др.) для сбора данных при экспорте.
- **LFO Store:** Визуализация и управление низкочастотным осциллятором.
- **Preset Store:** Сохранение и загрузка состояний всей студии (Snapshots).
- **MidiExporter:** Логика преобразования текущего состояния сессии в MIDI-файл.
- **Telegram WebApp:** Интеграция с API Telegram для отправки сгенерированных MIDI-файлов пользователю.

## 2. Интерфейс и UI Элементы

### Секция Master Transport
**Строки: 112-138**
Группа из трех регуляторов: BPM (60-200), Master Volume и Swing (степень "кача" ритма).

### Визуализатор LFO
**Строки: 27-48, 140-156**
Вращающийся 3D-додекаэдр, масштаб которого пульсирует в такт LFO. Отображает цель модуляции (например, Cutoff или Volume).

### Стойка эффектов и эквалайзера
**Строки: 176-240**
- **4-Band Master EQ:** Четыре регулятора для настройки частотного баланса всей композиции (Low, Lo-Mid, Hi-Mid, High).
- **FX Modules:** Три подсвеченных куба (Reverb, Delay, Distortion) с возможностью регулировки уровня Wet (количество эффекта).

### Панель пресетов и AI-генерации (Python Generate)
**Строки: 243-269**
- Кнопки быстрых пресетов (P1-P4).
- Кнопка "Python Random Preset", которая вызывает `CompositionManager` для генерации новой музыкальной идеи на основе алгоритмов.

## 3. Логика и Математика

### 3.1 Экспорт в MIDI

**Строки: 272-338**

Это одна из самых сложных логических цепочек компонента:
1. Собираются текущие паттерны всех инструментов из соответствующих сторов.
2. Данные передаются в `exportToMidi`, где происходит квантование и формирование MIDI-сообщений.
3. Результат конвертируется в Base64 и отправляется через POST-запрос на локальный сервер `/upload-midi`.
4. Взаимодействие сопровождается тактильной отдачей (`HapticFeedback`) через Telegram WebApp.

### 3.2 Визуализация LFO

**Строки: 32**

Математика трансформации значения LFO в визуальный масштаб:
$$Scale = 1 + LfoValue \times 0.5$$
Это создает плавную пульсацию объекта, которая визуально подтверждает работу модуляции.

### 3.3 Мастер Эквалайзер (Gain Mapping)

**Строки: 184-214**

Значения из регуляторов эквалайзера маппятся на диапазон $[-12, +12]$ дБ, что соответствует стандартным спецификациям профессиональных систем коррекции звука.
