# 📘 АУДИТ: HarmonyScreen.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент является центром управления многослойным синтезатором (Harmony Station), предназначенным для создания сложных текстур, пэдов и гармонических последовательностей. Дизайн выполнен в стиле "Amber Rack" (ретро-футуристичная рэковая стойка с янтарным дисплеем), имитирующем интерфейсы старых компьютеров или экспериментального лабораторного оборудования.

### Связи
- **Harm Store:** Управляет тремя осцилляторами, блоком Buchla 259, фильтр-секцией и FX-процессором.
- **Visual Store:** Обеспечивает выход из интерфейса (`SHUTDOWN`).
- **Tone.js Engine:** Все параметры из этого экрана напрямую транслируются в аудио-движок для изменения тембра в реальном времени.

---

## 2. Интерфейс и UI Элементы

### Структура синтезатора (Component Blocks)
1. **Oscillator Bank:** Группа из трех независимых осцилляторов с выбором формы волны (Sine, Saw, Square, Triangle) и возможностью расстройки (Detune) в центах и полутонах.
2. **Buchla 259 Core:** Секция вест-кост синтеза. Позволяет использовать "Wave Folding" (складывание волны) для получения богатых гармоник через параметры Timbre и Modulation Index.
3. **Analog Filter Strip:** Управляемый напряжением фильтр (VCF) с выбором типа (Lowpass, Highpass, Bandpass) и резонансом.
4. **FX Signal Processor:** Матрица эффектов, включающая Дисторшн, Фейзер, Дилей и Реверберацию.

## 3. Логика и Математика

### 3.1 Архитектура параметров

**Строки: 8-16**

Компонент использует деструктуризацию состояния `harmonyState` для чистого кода. Все изменения параметров проходят через единую функцию `setParam`, которая обновляет соответствующие узлы в аудио-движке.

### 3.2 Формирование тембра (Buchla Logic)

**Строки: 78-98**

Параметры секции Buchla работают по экспоненциальным или линейным кривым:
- **Timbre:** Управляет количеством "складок" в алгоритме Wavefolder.
- **FM/AM Index:** Глубина перекрестной модуляции между внутренними операторами.
$$Result_{wave} = \text{WaveFold}(\text{Oscillator}_{signal}, Timbre)$$

### 3.3 Визуальные эффекты интерфейса

Для создания атмосферы ретро-монитора используются CSS-слои:
- **Amber Grain:** Эффект зернистости пленки.
- **Scanlines:** Горизонтальные линии сканирования, характерные для ЭЛТ-мониторов.
- **Status Indicators:** Мигающие надписи "RUNNING" и ложные данные о напряжении/температуре для погружения в интерфейс.
