# 📘 АУДИТ: AcidSynth3D.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент представляет собой премиальную 3D-интерпретацию классического кислотного бас-синтезатора (стиль 303). Вместо имитации физических кнопок, дизайн выполнен в стиле "Glossy Magazine Edition" — минималистичное черное шасси с парящей над ним металлической каплей "жидкого" баса. Визуализация напрямую отражает характер звука: резонанс и частота среза фильтра (Cutoff) меняют физическую форму и скорость деформации капсулы.

### Связи
- **Bass Store:** Основной источник параметров звука (Resonance, Cutoff) и управления секвенсором (`isPlaying`).
- **Visual Store:** Интеграция с системой жестов для управления звуком через 3D-пространство.
- **Drei Ecosystem:** Использование продвинутых материалов `MeshDistortMaterial` и эффектов `ContactShadows`.

## 2. Интерфейс и UI Элементы

### Визуальный концепт
1. **Product Chassis:** Глянцевая подставка (`meshPhysicalMaterial` с `clearcoat`), создающая эффект дорогого студийного оборудования.
2. **Liquid Core:** Центральная металлическая сфера. Её деформация визуализирует "кислотный" характер звука — от спокойной зеркальной поверхности до хаотичных ртутных колебаний.
3. **Typography:** Парящие в воздухе надписи, выполненные в стиле люксовой электроники ("PURE RADIANCE").
4. **Action Button:** 3D-кнопка "ENGAGE/SUSPEND" для запуска и остановки басовой линии.

## 3. Logic & Interaction

### 3.1 Синхронизация формы и звука

**Строки: 27-35**

Физические свойства материала `MeshDistortMaterial` привязаны к музыкальным параметрам:
- **Resonance $\rightarrow$ Distort:** Чем выше резонанс фильтра, тем сильнее искажается сфера.
  $$Distort = \text{map}(Resonance_{0..20} \rightarrow 0.2..0.8)$$
- **Cutoff $\rightarrow$ Speed:** Частота среза управляет скоростью внутренних волн материала.
  $$Speed = \text{map}(\log_2(Cutoff)_{5..13} \rightarrow 2..6)$$

### 3.2 Пространственное управление (3D Drag)

**Строки: 38-44**

Пользователь может менять звук, перетаскивая сферу в 3D пространстве:
- **Перемещение по Y (Вертикаль):** Управляет частотой среза фильтра (Cutoff).
- **Перемещение по X (Горизонталь):** Управляет резонансом.
Математика преобразования перемещения учитывает инверсию и масштабирование, делая управление интуитивно понятным (движение вверх раскрывает фильтр).

### 3.3 Освещение и Тени

Компонент использует систему `spotLight` и `pointLight` для создания глубоких отражений на металле, а `ContactShadows` обеспечивает мягкое "приземление" объекта на невидимый пол студии.
