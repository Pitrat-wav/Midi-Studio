# 📘 АУДИТ: InstrumentNavigation.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует компактную панель навигации, расположенную в нижней части экрана. Она предоставляет пользователю быстрый доступ ко всем основным инструментам студии через иконки и текстовые метки. Панель служит визуальным дубликатом цифровых горячих клавиш (0-9).

### Связи
- **Store:** Обычно управляется через `useVisualStore` (фокусировка инструментов).
- **Navigation:** Синхронизирован с `KeyboardController.tsx`, отображая те же горячие клавиши и названия.
- **Стилизация:** Использует CSS-переменные для динамической раскраски кнопок в зависимости от выбранного инструмента.

---

## 2. Интерфейс и UI Элементы

### Структура кнопки навигации
Каждая кнопка в панели содержит:
- **Icon:** Эмодзи, символизирующий тип инструмента.
- **Label:** Краткое текстовое название.
- **Shortcut:** Номер горячей клавиши (0-9) для обучения пользователя.
- **Color Indicator:** Индивидуальный цвет инструмента, передаваемый через CSS-переменную `--button-color`.

### Список доступных инструментов
Панель включает 10 позиций:
1. **All (🌌):** Общий вид всей 3D студии.
2. **Drums (🥁):** Драм-машина.
3. **Bass (🎸):** Басовый синтезатор.
4. **Synth (🎹):** Модульный синтезатор Harmony.
5. **Pads (☁️):** Эмбиент-пэды.
6. **Seq (🎛️):** Секвенсор ML-185.
7. **Drone (☄️):** Дроун-станция.
8. **Master (🕹️):** Мастер-канал и эффекты.
9. **Sampler (🎚️):** Гранулярный сэмплер Fairlight.
10. **Buchla (🧬):** Сложный осциллятор Buchla 259.

## 3. Логика и Математика

### 3.1 Управление состоянием активности

**Строки:** 32-34

Компонент определяет состояние текущей кнопки через простое сравнение `currentInstrument === instrument.id`. Если условие истинно, кнопке добавляется CSS-класс `.active`, что вызывает анимацию масштабирования и свечения.

### 3.2 Динамическая стилизация (CSS Variables)

**Строки:** 40-42

Для того чтобы кнопки имели разный цвет (красный для барабанов, синий для баса и т.д.), используется передача переменной напрямую в стиль React-компонента:
```javascript
style={{ '--button-color': instrument.color } as React.CSSProperties}
```
Это позволяет избежать громоздких условных классов в CSS и централизованно управлять цветовой палитрой в массиве `INSTRUMENTS`.
