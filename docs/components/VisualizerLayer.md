## 1. Глобальное Описание

### Назначение
`VisualizerLayer.tsx` — это слой-контейнер для полноэкранных аудио-визуализаторов в режиме `VISUALIZER`. Он отвечает за окружение (звезды, свет, туман) и переключение между различными алгоритмами визуализации (`FeedbackNebula`, `GeometricDynamic`).

### Связи
- **Store:** Использует `visualStore` для получения `visualizerIndex` и `globalAudioIntensity`.
- **Drei:** Интеграция компонентов `Stars`, `Float`, `Center` для улучшения эстетики.

---

## 2. Интерфейс и UI Элементы (3D)

### Компоненты окружения
- **Stars:** Создает эффект глубокого космоса вокруг визуализатора.
- **Float/Center:** Обеспечивает плавное парение и центровку визуальных объектов в пространстве.
- **Fog:** Используется для создания глубины и плавного растворения объектов на горизонте.
