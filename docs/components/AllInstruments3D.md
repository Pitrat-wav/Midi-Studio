# 📘 АУДИТ: AllInstruments3D.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент является главным оркестратором 3D-мира студии. Он собирает все отдельные инструменты (DrumMachine, AcidSynth, Sampler и др.) в единую композицию, расставляя их в пространстве согласно конфигурации `SpatialLayout`. Также компонент управляет глобальным реактивным освещением и реализует уникальный режим **South Park**, заменяя или дополняя 3D-модели персонажами мультсериала.

### Связи
- **Spatial Layout:** Определяет точные координаты $[x, y, z]$ для каждого инструмента.
- **AudioVisual Bridge:** Передает данные о частотах в систему реактивного освещения (`ReactiveLights`).
- **Visual Store:** Переключает визуальные темы и управляет фокусом.
- **Instruments:** Импортирует и инициализирует все 14 подсистем визуализации оборудования.

## 2. Интерфейс и UI Элементы

### Архитектура группы инструментов
**Строки: 33-64**
Каждый инструмент обернут в `InstrumentGroup`, которая перехватывает клики для фокусировки камеры. Если инструмент уже находится в фокусе, клик по нему сбрасывает камеру в общий обзор.

### Режим South Park (Billboard Rendering)
**Строки: 67-104**
При активации темы `southpark`, рядом с каждым синтезатором появляется "персонаж-куратор" (например, Картман управляет барабанами, Кенни — басом). Персонажи реализованы как билборды (плоские спрайты, всегда повернутые к камере), которые пульсируют и деформируются под ритм своего инструмента.

## 3. Логика и Математика

### 3.1 Реактивное освещение (Reactive Lights)

**Строки: 159-198**

Система освещения студии динамически меняется под музыку:
1. **Directional Light (Основной свет):** Интенсивность коррелирует с мощностью низких частот (ударами бочки).
   $$Intensity_{dir} = 0.5 + (uLowFreq \times 0.5)$$
2. **Ambient Light (Фоновый свет):** Повышает общую освещенность сцены при увеличении общей громкости.
   $$Intensity_{amb} = 0.2 + (uAudioIntensity \times 0.2)$$
3. **Hemisphere Light (Небо):** Меняет оттенок (Hue) при появлении высоких частот (тарелки, хай-хэты).
   $$Hue = (uHighFreq \times 0.2) \pmod 1$$

### 3.2 Пространственное разделение

Инструменты разделяются по функциональным кластерам:
- **Core Loop:** Барабаны, Бас, Пэды, Гармония.
- **Sequencer Cluster:** Линейные и змейковидные секвенсоры.
- **Master Section:** Пульт управления, Микшер и Клавиатура.
