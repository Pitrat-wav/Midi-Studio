# VisualizerSearch.tsx

### 1. Глобальное Описание (File Overview)
* **Назначение:** Всплывающий поисковый интерфейс (Command Palette) для быстрого выбора визуализаторов по названию или тегам.
* **Связи:** 
    * Импортирует: `useVisualStore`, `VISUALIZER_REGISTRY`.

### 2. Интерфейс и UI Элементы (Visual Elements)

* **Элемент:** Поле ввода (Search Input)
    * **Где находится:** Строки 91-109.
    * **Логика:** Фокусируется автоматически при открытии. Фильтрует список в реальном времени.

* **Элемент:** Список результатов (Results List)
    * **Где находится:** Строки 113-151.
    * **Логика:** Отображает иконку, имя и теги визуализатора.

### 3. Логика и Математика (Logic & Math)

* **Функция: Фильтрация (`results`)**
* **Логика:** Использует `Array.filter` для поиска вхождения строки запроса.
* **Математическая модель:**
    $$ Match = \text{RegExp}(Query) \in (Name \cup Tags) $$

* **Построчный разбор (`handleKeyDown`):**
    1. Проверяет клавишу `/` для открытия поиска (только в режиме VISUALIZER).
    2. Клавиша `ArrowDown`/`ArrowUp`: Изменяет `selectedIndex` по модулю длины массива результатов:
       $$ Index_{next} = (Index \pm 1 + N) \pmod N $$
    3. Клавиша `Enter`: Вызывает `setVisualizerIndex` для выбранного ID и закрывает поиск.
