# 📘 АУДИТ: TabNavigation.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует нижнюю панель навигации (Bottom Tab Bar), оптимизированную для мобильных устройств и интеграции с **Telegram WebApp**. Она позволяет быстро переключаться между основными функциональными разделами приложения: драм-машиной, басом, секвенсором, пэдами, гармонией и разделом помощи.

### Связи
- **Navigation:** Управляет состоянием `activeTab`, которое определяет, какой экран (HUD) отображается пользователю.
- **Styling:** Использует системные переменные Telegram (`tg-theme`) и эффекты матового стекла (`glass-bg`, `backdrop-filter`).

---

## 2. Интерфейс и UI Элементы

### Дизайн вкладок
Каждый элемент навигации состоит из:
- **Icon:** Векторная иконка из набора `lucide-react`, визуально представляющая раздел.
- **Label:** Краткая текстовая подпись.
- **Active State:** Выделение цветом темы кнопки (`button-color`) и легкое масштабирование иконки (`scale(1.1)`) для обратной связи.

### Адаптивность
- **Safe Area:** Учитывает отступ снизу (`env(safe-area-inset-bottom)`) для корректного отображения на современных смартфонах с "челками" и жестовой навигацией.
- **Layout:** Равномерное распределение кнопок по всей ширине экрана.

---

## 3. Логика и Математика

### 3.1 Расчет визуальной иерархии

Компонент использует простое условие `activeTab === tab.id` для динамического изменения стилей. 
**Цветовая логика:**
- Active: `var(--tg-theme-button-color)` (Главный акцентный цвет бота).
- Inactive: `var(--tg-theme-hint-color)` (Приглушенный серый).

**Трансформации:**
При активации вкладки применяется интерполированный переход:
$$Scale = isActive ? 1.1 : 1.0$$
$$StrokeWidth = isActive ? 2.5 : 2.0$$
Это создает ощущение "живого" интерфейса при переключении разделов.
