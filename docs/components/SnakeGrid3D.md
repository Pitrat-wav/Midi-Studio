# 📘 АУДИТ: SnakeGrid3D.tsx

## 1. Глобальное Описание

### Назначение
Компонент `SnakeGrid3D` визуализирует работу пространственного секвенсора MDD Snake. Это сетка 4x4, по которой перемещается "голова" алгоритма, выбирая ноты на основе заданных правил навигации. Визуализация в 3D позволяет пользователю видеть не только текущую нажатую ноту, но и всю структуру сетки с выделенными активными областями.

### Связи
- **Sequencer Store:** Содержит массив `snakeGrid` (данные о нотах в ячейках) и `currentSnakeIndex` (позиция "головы").
- **Spatial Layout:** Позиционирует сетку в зоне логических инструментов студии.

## 2. Интерфейс и UI Элементы

### Сетка ячеек (4x4 Grid)
**Строки: 16-32**
- **Ячейки:** Плоские прозрачные боксы.
- **Индикация состояний:**
    - **Активная голова (Active Head):** Ярко-розовое свечение (`#ff00ff`).
    - **Наполненная ячейка (Filled Cell):** Темно-фиолетовый цвет, если в ячейке есть нота.
    - **Пустая ячейка:** Почти прозрачный темно-серый фон.

## 3. Логика и Математика

### 3.1 Расчет координат сетки

**Строки: 17-18**

Индекс массива $[0..15]$ преобразуется в 2D координаты по осям X и Z:
$$X = (Index \pmod 4) - 1.5$$
$$Z = \lfloor \frac{Index}{4} \rfloor - 1.5$$
Смещение на $-1.5$ необходимо для центрирования сетки 4x4 вокруг нулевой точки (чтобы ячейки шли от -1.5 до +1.5 с шагом 1).
