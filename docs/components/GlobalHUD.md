# 📘 АУДИТ: GlobalHUD.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент представляет собой высокоуровневый графический оверлей (2D HUD), работающий поверх 3D сцены. Он обеспечивает управление критическими системными функциями: активацию AI-зрения (Hand Tracking), управление микрофоном, переключение визуальных тем и отображение статусных сообщений. **GlobalHUD** — это мост между глубокими настройками движка и пользователем, оформленный в футуристичном полупрозрачном стиле.

### Связи
- **Visual Store:** Считывает и обновляет темы (`aestheticTheme`), пресеты фона и состояние трекинга рук.
- **Audio Store:** Управляет состоянием микрофона и мониторинга, получает текущие значения BPM и громкости.
- **Gesture Manager:** Получает данные о жестах по краям экрана для отображения индикаторов Master Volume и BPM.

## 2. Интерфейс и UI Элементы

### Системная панель (Top Right)
**Строки:** 21-55
Расположена в верхнем правом углу и содержит:
- **Vision Toggle:** Переключатель трекинга рук.
- **Mic Control:** Кнопки включения микрофона и мониторинга (появляются контекстно).
- **Whisk UI:** Меню выбора визуальных стилей.

### Смена тем (Visual Themes)
**Строки:** 146-335
Компактная сетка пресетов с уникальной стилизацией для каждой кнопки:
- **Default:** Оранжево-желтый градиент.
- **Cosmic:** Розово-голубой неон.
- **Cyber:** Зелено-золотой киберпанк.
- **South Park Rock:** Специальная кнопка с градиентом природы и неба, активирующая 2.5D режим.

## 3. Логика и Математика

### 3.1 Индикаторы жестов (Edge Swipes)

**Строки:** 81-99

Когда пользователь выполняет жест по краю экрана, HUD отображает крупный всплывающий индикатор в центре.
**Логика отображения:**
- Если `edgeSide === 'top'`, выводится Master Volume.
- Если `edgeSide === 'bottom'`, выводится Global BPM.
Это обеспечивает визуальную синхронизацию с физическим движением пальца.

### 3.2 Управление визуальными пресетами

**Строки:** 150-160

Компонент находит индексы пресетов в массиве `PRESETS` по их именам (`WHISK: COSMIC` и т.д.). При нажатии на кнопку вызывается `setAestheticTheme` и `setBackgroundPreset`, что инициирует мгновенную перестройку 3D окружения и шейдеров.

### 3.3 Стилизация (Glassmorphism)

Применяется продвинутый CSS для создания эффекта матового стекла:
- `backdropFilter: 'blur(15px)'` — размытие заднего 3D плана.
- `background: 'linear-gradient(...)'` — глубокие синие тона с прозрачностью.
- `boxShadow: 'inset 0 0 30px ...'` — внутреннее неоновое свечение.
