# 📘 АУДИТ: Knob.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует **Поворотный Регулятор (Knob)** — основной элемент пользовательского интерфейса для управления непрерывными параметрами звука (громкость, частота среза, глубина эффектов). Он имитирует поведение физических ручек аналоговых синтезаторов, позволяя пользователю изменять значения путем вертикального перетаскивания (драга).

### Связи
- **Store:** Использует `visualStore` для передачи данных о взаимодействии, что может влиять на визуальные эффекты в 3D сцене.
- **Платформа:** Интегрирован с Telegram WebApp API для обеспечения тактильной отдачи (Haptic Feedback) при изменении значений.
- **Использование:** Применяется повсеместно во всех экранах управления инструментами (Drums, Bass, Harm и др.).

---

## 2. Интерфейс и UI Элементы

### Визуальные компоненты
* **SVG Кольцо (Progress Ring):** Визуализирует текущее значение параметра. Состоит из фонового кольца (135° - 405°) и активного сегмента.
* **SVG Индикатор (Pointer):** Линия на "крышке" ручки, указывающая на текущее значение.
* **Label (Метка):** Текстовое название параметра.
* **Value Display:** Цифровое отображение текущего значения в моноширинном шрифте.

## 3. Логика и Математика

### 3.1 Расчет значения при перетаскивании (Vertical Drag)

**Строки:** 39-60

В отличие от вращения, этот регулятор использует вертикальное движение пальца/мыши для изменения значения, что более удобно на мобильных устройствах.

**Математическая модель:**
$$Value_{new} = Value_{start} + \frac{\Delta y}{Sensitivity} \times (Max - Min)$$

Где:
- $\Delta y$: Разница между начальной и текущей координатой Y (инвертированная).
- $Sensitivity$: Чувствительность (установлена в 240 пикселей для полного оборота).
- Результат ограничивается диапазоном $[Min, Max]$.

### 3.2 Квантование (Step)

Для параметров, требующих дискретных значений (например, выбор типа волны), применяется округление:
$$Value_{final} = \text{round}\left(\frac{Value_{new}}{Step}\right) \times Step$$

---

### 3.3 Визуальное представление (SVG)

**Строки:** 79-84

Для отрисовки дуги используется свойство `strokeDashoffset`.

**Математика дуги:**
- Угол поворота индикатора: $Angle = NormalizedValue \times 270^\circ - 135^\circ$
- Смещение дуги: $Offset = Circumference \times (1 - NormalizedValue \times 0.75)$
- Множитель 0.75 ограничивает дугу до 270 градусов (три четверти круга).

---

### 3.4 Взаимодействие и обратная связь

- **Single Click/Drag:** Изменение значения + `selectionChanged` (Haptic).
- **Double Click:** Возврат к значению по умолчанию (Default Value) или к середине диапазона + `notificationOccurred('success')`.
- **Pointer Capture:** Используется `setPointerCapture` для удержания фокуса на ручке, даже если палец вышел за её пределы во время движения.
