# 📘 АУДИТ: RadialMenu3D.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент реализует интерактивное круговое (радиальное) меню, которое появляется в 3D пространстве в точке клика или долгого нажатия. Этот тип интерфейса идеально подходит для VR/AR и пространственных приложений, так как позволяет осуществлять выбор действия быстрым движением курсора или пальца в любом направлении. В приложении оно используется для контекстных команд (Focus, Overview, Panic).

### Связи
- **Gesture Manager:** Вызывается жестом "Long Press".
- **WebGL Scene:** Отрисовывается как оверлей внутри 3D холста.

---

## 2. Интерфейс и UI Элементы

### Визуальная структура
- **Background Ring:** Полупрозрачное темное кольцо (`Ring`), служащее подложкой для пунктов меню.
- **Action Nodes:** Круглые цветные кнопки, расположенные по периметру. Каждая кнопка имеет свой цвет и привязана к определенной команде.
- **Dynamic Text:** Текстовые метки под каждой кнопкой, поясняющие действие.

## 3. Логика и Математика

### 3.1 Плавное появление (Lerp Animation)

**Строки: 34-42**

Меню не появляется мгновенно, а плавно "вырастает" из точки:
**Масштабирование:**
- Если `visible == true`: $Scale \rightarrow 1.0$ (коэффициент 0.1).
- Если `visible == false`: $Scale \rightarrow 0.0$ (коэффициент 0.2).
Также добавлено легкое вращение по оси Z (`rotation.z += 0.01`), что придает меню эффект парения.

### 3.2 Радиальное распределение

**Строки: 54-57**

Координаты пунктов меню рассчитываются на основе их индекса:
$$\text{Angle}_i = \frac{i}{N} \times 2\pi$$
$$X = \cos(\text{Angle}_i) \times 1.5$$
$$Y = \sin(\text{Angle}_i) \times 1.5$$
Это гарантирует равномерное распределение любого количества элементов по окружности радиусом 1.5.
