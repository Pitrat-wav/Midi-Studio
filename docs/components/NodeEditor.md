# 📘 АУДИТ: NodeEditor.tsx

## 1. Глобальное Описание

### Назначение
Этот компонент представляет собой полнофункциональную модульную среду (Modular Environment), построенную на базе `ReactFlow`. Он позволяет пользователю создавать сложные аудио-тракты, логические схемы и системы обработки MIDI-событий путем визуального программирования. **THE GRID** — это "сердце" гибкости приложения, где жесткая структура инструментов сменяется полной свободой творчества.

### Связи
- **ReactFlow:** Основной движок для отрисовки графа, управления узлами (nodes) и связями (edges).
- **Node Store:** Синхронизирует состояние визуального графа с реальными аудио-объектами в `GraphEngine.ts`.
- **Custom Nodes:** Использует `CustomNode.tsx` для отображения каждого блока.

## 2. Интерфейс и UI Элементы

### Sidebar Palette (Боковая панель)
Список доступных модулей, разделенный на категории:
- **Audio (Stereo):** Осцилляторы, фильтры, смесители.
- **Logic:** Клоки, математические операции, рандомизаторы.
- **Note / MIDI:** Квантайзеры, аккордовые генераторы, арпеджиаторы.
- **FX Rack:** Эффекты задержки, реверберации и искажения.
- **Advanced:** Редактор скриптов (JSFX), осциллограф и AI-агенты.

### Главный холст (Canvas)
- **Background:** Темная сетка для удобного позиционирования.
- **Minimap:** Мини-карта для навигации по большим схемам (узлы подсвечены цветом категории).
- **ContextMenu:** Всплывающее меню для быстрых действий (Duplicate, Delete).
- **Controls:** Кнопки масштабирования и центрирования вида.

## 3. Logic & Interaction

### 3.1 Drag and Drop (D&D)

**Строки:** 63-75, 201-213

Реализована нативная поддержка Drag & Drop для добавления новых узлов:
1. При начале перетаскивания из сайдбара (`onDragStart`), тип узла записывается в `dataTransfer`.
2. При отпускании на холсте (`onDrop`), координаты события преобразуются в координаты графа, и вызывается `addNode`.

### 3.2 Контекстное управление (Contexts)

**Строки:** 86-99

Переключатель **POLY / FX / NOTE** меняет режим работы всего графа:
- **POLY:** Создание монофонических или полифонических синтезаторов.
- **FX:** Проектирование систем обработки звука.
- **NOTE:** Работа с логикой генерации нот.

### 3.3 Манипуляции с графом

**Строки:** 44-61

Функции `onDuplicate` и `onDelete` обеспечивают управление жизненным циклом узлов. Дублирование создает точную копию узла с небольшим смещением координат $(x+20, y+20)$ для визуального разделения.
