# 📘 GENESIS DOC: VisualEngine.tsx

## 1. Глобальное Описание

### Назначение
`VisualEngine` — это выделенный, высокопроизводительный графический движок для создания иммерсивных аудио-визуальных ландшафтов. В отличие от основной 3D-студии, этот движок сфокусирован исключительно на генеративном искусстве и шейдерных эффектах ("TouchDesigner style"), отключая все элементы интерфейса и инструменты для максимизации ресурсов GPU.

### Связи
- **App.tsx:** Является основным видом приложения при активации режима `VISUALIZER`.
- **visualStore:** Синхронизирует индекс текущего визуализатора и интенсивность аудио.

---

## 2. Логика и Математика

### 2.1 Архитектура переключения
Движок использует `Suspense` и условный рендеринг для мгновенной смены алгоритмов. Каждый визуализатор является полностью автономным модулем.

### 2.2 Рендеринг (Canvas)
Используется оптимизированный `Canvas` с отключенным антиалиасингом для шейдерно-тяжелых эффектов:
```tsx
gl={{ antialias: false, powerPreference: 'high-performance' }}
```
