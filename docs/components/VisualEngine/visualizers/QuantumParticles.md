# 📘 GENESIS DOC: QuantumParticles.tsx (Engine Edition)

## 1. Глобальное Описание

### Назначение
`QuantumParticles` — это высокопроизводительная система частиц (50,000 точек), формирующая динамическое квантовое облако. Она визуализирует "дыхание" звукового спектра через пространственную деформацию и изменение плотности точек.

---

## 2. Логика и Математика

### 2.1 Spatial Deformation (Пространственная деформация)
Каждая частица имеет начальную позицию на сфере. В реальном времени позиция пересчитывается с учетом шума и данных FFT:
$$Position_{t} = Position_{init} \times (1 + Noise + FFT\_Scale)$$
Где `FFT_Scale` привязан к конкретной частоте для каждой точки.

### 2.2 Additive Blending
Используется режим смешивания `AdditiveBlending`, что позволяет точкам "накладываться" друг на друга, создавая эффект ярких энергетических сгустков при высокой плотности.
