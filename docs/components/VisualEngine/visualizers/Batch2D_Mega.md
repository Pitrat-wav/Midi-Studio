# Batch2D_Mega.tsx

### 1. Глобальное Описание (File Overview)
* **Назначение:** Коллекция "легких" 2D визуализаторов (ID 61-160), оптимизированных для работы в большом количестве. Включает несколько уникальных реализаций и фабрику для генерации процедурных вариаций.
* **Связи:** 
    * Импортирует: `useVisualStore` (для аудио-интенсивности).
    * Используется в: `VisualizerRegistry` (импортируется как пак).

### 2. Интерфейс и UI Элементы (Visual Elements)
* **Элемент:** Канвас (Three.js Scene)
* **Логика:** Визуализаторы рендерятся внутри R3F Canvas. Не имеют собственного HTML UI, реагируют только на аудио.

### 3. Логика и Математика (Logic & Math)

* **Функция:** `Base2D`
* **Что делает:** Обертка `group` для стандартизации трансформов.

* **Функция:** `CyberRain` (ID 62)
* **Логика:** Система частиц (200 шт), падающих вниз. При пересечении Y < -5 переносятся наверх (Y = 5).
* **Математика:**
    $$ Y_{new} = Y_{old} - (speed \times (1 + Intensity)) $$

* **Функция:** `createVisualizer(id, name, color, type)`
* **Что делает:** Фабрика компонентов (Higher-Order Component generator). Создает компонент визуализатора на основе параметров.
* **Типы:**
    * `blob`: Пульсирующий круг/фигура.
        * $$ Scale = 1 + \sin(2t) \times 0.2 + Intensity \times 0.5 $$
        * $$ Rotation = t \times 0.01 $$
    * `lines`: Движущиеся горизонтальные линии.
        * $$ X = \sin(t) \times Intensity $$
    * `grid`: Сетка `gridHelper`.
    * `shapes`: Вращающийся каркас.

### 4. Реализованные Визуализаторы (Partial List)
* **Neon Horizon (61):** Сетка в перспективе + пульсирующее солнце.
* **Digital Pulse (63):** Расширяющееся кольцо.
* **Vortex 2D (64):** Вращающийся спиральный массив плоскостей.
* **Batch Generative (65-160):** 95 процедурно сгенерированных вариаций (Star Field, Lava Flow, Matrix, и т.д.).
