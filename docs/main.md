# main.tsx

### 1. Глобальное Описание (File Overview)
* **Назначение:** Точка монтирования React-приложения в DOM-дерево. Оборачивает приложение в StrictMode и глобальный ErrorBoundary.
* **Связи:** 
    * Импортирует: `App`, `ErrorBoundary`, глобальные стили `index.css`.
    * Монтируется в: `document.getElementById('root')`.

### 2. Интерфейс (Visual Elements)
* **Элемент:** React Root
    * **Логика:** Рендерит дерево компонентов внутри `div#root`.

### 3. Логика и Математика (Logic & Math)

* **Построчный разбор:**
    1. Создаем `Root` через `ReactDOM.createRoot`.
    2. Используем `React.StrictMode` для выявления потенциальных проблем (двойной рендер в dev-режиме).
    3. Оборачиваем все в `ErrorBoundary`, чтобы перехватывать фатальные ошибки рендеринга на самом верхнем уровне.
