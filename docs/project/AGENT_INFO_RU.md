# 🤖 Информация о Проекте (Agent Info)

## 📦 Studio Elite 3D (v5.0.0)
Автономная генеративная 3D аудио-станция.

### 🗺 Карта Кодовой Базы

| Директория | Ответственность |
|:---|:---|
| `src/logic` | Аудио-движки, секвенсоры, математические алгоритмы. |
| `src/store` | Zustand контейнеры состояния (Audio, Visual, Themes). |
| `src/components/WebGL` | Ядро 3D сцены, освещение, визуальные эффекты. |
| `src/components/WebGL/instruments` | Реализация 3D станций (Drums, Bass, etc.). |
| `src/lib` | Утилиты (AudioVisualBridge, GamepadManager). |

### 🚀 Ключевые Вехи (Milestones)

1.  **3D Migration (v3.0)**: Полный переход от гибридного интерфейса к чистому WebGL пространству.
2.  **Node Editor (v4.5)**: Внедрение модульного графа для кастомного синтеза.
3.  **Arrangement 2.0 (v4.4)**: Исправление таймлайна, клипов и UX аранжировки.
4.  **Agentic Update (v5.0)**:
    - Интеграция геймпада PS5.
    - Динамическая система тем (Prism).
    - Полная русская документация.

### 🧠 Алгоритмы
- **Euclidean**: Ритмическое распределение (Bjorklund).
- **Sting (Acid)**: Вероятностная генерация баса.
- **MDD Snake**: Пространственный секвенсор (Snake Patterns).

### 🛡 Принципы Агента
- **Код**: TypeScript, строгая типизация.
- **Стиль**: Functional React, Hooks, Zustand.
- **Доки**: Поддержание актуальности документации на русском языке.
