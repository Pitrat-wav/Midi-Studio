# 🌌 Руководство Пользователя Studio Elite 5.0

Добро пожаловать в **Telegram MIDI Studio Pro 3D** (Studio Elite) — автономную генеративную рабочую станцию нового поколения.

Это руководство подробно описывает каждый элемент интерфейса, каждую ручку и каждую функцию системы.

---

## 🎮 Управление (Навигация)

Студия представляет собой полностью трехмерное пространство.

### ⌨️ Клавиатура и Мышь
- **WASD / Стрелки**: Перемещение камеры (полет).
- **ПКМ (Правая кнопка) + Драг**: Вращение камеры (Осмотр).
- **ЛКМ (Клик)** по инструменту: Фокусировка (Приближение).
- **Двойной Клик**: Мгновенный переход к инструменту.
- **Пробел**: Play / Pause.
- **H**: Скрыть/Показать интерфейс (HUD).
- **?**: Вызов справки.

### 🕹️ Геймпад (PlayStation 5 / Xbox) — **НОВОЕ**
- **Левый Стик**: Перемещение (Ходьба).
- **Правый Стик**: Вращение камеры (Осмотр).
- **L1 / R1**: Переключение между инструментами (Цикл).
- **Крест (A)**: Play / Stop.
- **Круг (B)**: Mute (Заглушить) текущий инструмент.
- **Квадрат (X)**: Режим обзора (Overview).
- **Треугольник (Y)**: PANIC (Остановить весь звук).
- **D-Pad ⬆️**: Смена фона (Background).
- **D-Pad ⬅️/➡️**: Изменение темпа (BPM).

## 🎹 Инструменты (Модули)

В студии работает 8 независимых аудио-движков.

### 1. 🥁 Drums (Ударные)
Эвклидовый ритм-генератор.
- **Kick / Snare / HiHat**: Основные барабаны.
- **Steps**: Количество шагов в цикле.
- **Pulses**: Количество ударов (распределяются по эвклидовому алгоритму).
- **Visual**: Голографические кольца.

### 2. 🎸 Bass (Бас)
Кислотный синтезатор (TB-303 Clone).
- **Cutoff**: Частота среза фильтра.
- **Resonance**: "Визг" фильтра.
- **Env Mod**: Влияние огибающей на фильтр.
- **Visual**: Жидкий металл (Ртуть).

### 3. 🎼 Harmony (Гармония)
Полифонический синтезатор пэдов и лидов.
- **Type**: Выбор волны (Saw, Square, Sine).
- **Chord**: Тип аккорда (Minor, Major, 7th).
- **Smart Chord**: Автоматическая гармонизация одной ноты.

### 4. 🎛️ Buchla (Комплексный Генератор)
Эмуляция West Coast синтеза.
- **Timbre**: Волшебная ручка "Wavefolding" (насыщение гармониками).
- **FM Index**: Частотная модуляция.
- **Visual**: Абстрактная геометрия.

### 5. 📼 Sampler (Fairlight CMI)
Ретро-семплер из 80-х.
- **Page-R**: Секвенсор паттернов на зеленом CRT-экране.
- **Controls**: Grain (Зерно), Speed (Скорость).
- **Visual**: Винтажный терминал.

## 🎼 Режим Аранжировки (Arrangement Mode)

Полноценная DAW для сборки треков.

### Интерфейс
- **Треки**: Слева расположен список дорожек с мини-микшером (Volume/Mute/Solo).
- **Таймлайн**: Справа — сетка времени.
- **Клипы**: Цветные блоки с аудио-данными.

### Инструменты
- **Карандаш (Draw)**: Рисование автоматизации.
- **Ножницы (S)**: Разрезание клипов.
- **Дубликация (Cmd+D)**: Копирование выбранного.
- **Loop**: Зацикливание выбранного региона.

### Темы (Theme Engine)
В верхней панели находится выпадающий список тем и кнопка "⚡" (Shuffle).
- Выбирайте из **50 пресетов** (Cyber, Nature, Dark, Light).
- Темы меняют цвета интерфейса, фон 3D сцены и туман.

## 🧠 Редактор Нод (Node Editor)

Модульная среда для создания кастомных синтезаторов.
- **ПКМ (Правая кнопка)**: Открыть меню создания нод.
- **Категории**:
  - **IO**: Входы/Выходы (Audio Output, MIDI In).
  - **OSC**: Генераторы (Sine, Saw, Pulse).
  - **FX**: Эффекты (Reverb, Delay, Distortion).
  - **MATH**: Математика сигналов (Add, Mul).
- **Соединение**: Тяните провода от выхода к входу.

## 🛠 Техническая Информация
- **Движок**: Tone.js (Audio) + Three.js (Visuals).
- **Точность**: 64-bit floating point audio.
- **Задержка**: <10ms (в зависимости от буфера).
