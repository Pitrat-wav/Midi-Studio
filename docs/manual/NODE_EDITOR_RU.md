# 🎛️ Руководство по Модульному Редактору Нод (Node Editor)

Добро пожаловать в **Node Grid** — сердце процедурного звука в MIDI Studio Pro. Здесь вы можете создавать собственные синтезаторы, эффекты и генеративные патчи, соединяя модули проводами.

---

## 🚀 Основы (The Basics)

### Как открыть?
Нажмите на иконку **"Сетка"** (набор квадратов) или клавишу `Tab` (если настроено), чтобы войти в режим редактора нод.

### Палитра Модулей
Слева находится палитра с категориями:
- **AUDIO**: Генераторы звука (Oscillators), Фильтры, Эффекты.
- **LOGIC**: Математика, Клоки, Секвенсоры.
- **NOTE**: Работа с высотой тона и MIDI.
- **IO**: Ввод/Вывод (Audio Out, MIDI In).
- **ADVANCED**: Скрипты и AI.

Перетащите (Drag & Drop) модуль на сетку, чтобы создать его.

---

## 🔗 Соединения (Wiring)

Звук и данные текут слева направо.
- **Выходы** (справа на ноде) -> **Входы** (слева на ноде).
- **Цвета проводов**:
  - 🔵 **Audio**: Звуковой сигнал (стерео/моно).
  - 🟠 **Signal**: Управляющий сигнал (CV, LFO, Envelopes).
  - 🟣 **Data**: Сложные данные (массивы, объекты).

> **💡 Совет**: Вы можете модулировать почти любой параметр! Попробуйте подключить `LFO` к входу `Freq` у `Oscillator` для создания вибрато.

## 📚 Библиотека Модулей

### 🔊 Audio (Звук)
- **Oscillator**: Основа синтеза. Волны: *Sine, Square, Saw, Triangle*. Имеет вход FM для частотной модуляции.
- **Filter**: Срезает частоты. *Lowpass, Highpass*. Вход Cutoff управляет частотой среза.
- **Audio Env (ADSR)**: Огибающая громкости. Требует сигнал **Gate** для запуска.
- **Effects**: Delay, Reverb, Distortion.

### 🧠 Logic & Control (Логика)
- **Clock**: Генератор импульсов (BPM). Нужен для запуска секвенсоров.
- **Sequencer (8-Step)**: Выдает последовательность значений (CV). Имеет встроенные слайдеры для настройки шагов.
- **Quantizer**: Привязывает произвольный сигнал к музыкальным нотам (полутонам).
- **Math**: Сложение, умножение сигналов.

### 🎹 MIDI & IO
- **MIDI In**: Получает сигналы с вашей клавиатуры.
  - **Note**: Высота тона (0.0 - 1.0).
  - **Gate**: Нажатие клавиши (0 или 1).
  - **Vel**: Сила нажатия.
- **Audio Out**: Финальный выход на динамики. **Обязателен** для того, чтобы услышать звук!

### 🔮 Advanced (Pro)
- **Euclidean Sequencer**: Генератор ритмов на основе алгоритма Бьорклунда.
  - `steps`: Длина цикла.
  - `pulses`: Количество ударов.
  - `rotate`: Сдвиг паттерна.
- **Oscilloscope**: Визуализация сигнала в реальном времени. Подключите любой выход к `in`.
- **AI Gen**: Триггер нейросети.
  - При поступлении сигнала (> 0.5) отправляет запрос на генерацию текстуры для выделенного объекта.
  - Требует настройки API ключа (см. AI Guide).

## 🛠️ Сценарии использования (Cookbook)

### 3. Эвклидовый Ритм
1. **Clock** -> **Euclidean Seq**.
2. **Euclidean Seq** -> **Audio Env** (Gate).
3. **Oscillator** -> **VCA**.
4. Результат: Сложный полиритмический рисунок.


### 1. Простой Синтезатор
1. Создайте **MIDI In**.
2. Создайте **Oscillator**. Подключите `Note` (из MIDI) -> `Freq` (в Osc).
3. Создайте **Audio Env**. Подключите `Gate` (из MIDI) -> `Gate` (в Env).
4. Создайте **VCA** (Amplifier).
   - Подключите `Out` (Osc) -> `In` (VCA).
   - Подключите `Out` (Env) -> `CV` (VCA).
5. Подключите `Out` (VCA) -> **Audio Out**.

### 2. Генеративное Техно
1. Создайте **Clock**.
2. Создайте **Sequencer**. Подключите `Pulse` (Clock) -> `Clock` (Seq).
3. Настройте слайдеры на секвенсоре.
4. Создайте **Oscillator**. Подключите `CV` (Seq) -> `Freq` (Osc).
5. Добавьте **Delay** для атмосферы.
6. Выведите на **Audio Out**.

## 👨‍💻 Программирование (Script Node)
Для продвинутых пользователей доступна нода **Script JS**.
- Пишите DSP-код на JavaScript прямо в браузере.
- Функция `process(inputs, output)` вызывается для каждого аудио-буфера.
- Доступен объект `memory` для сохранения состояния между кадрами.

```javascript
// Пример: Простой Gain
function process(inputs, output) {
  for (let i = 0; i < inputs[0].length; i++) {
    output[i] = inputs[0][i] * 0.5; // Громкость 50%
  }
}
```
