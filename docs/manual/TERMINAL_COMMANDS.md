# Live Coding Terminal Commands

Консоль (Hacker Mode) позволяет управлять параметрами студии в реальном времени с помощью текста. 

## Как открыть
Нажмите клавишу **`** (тильда) на клавиатуре.

## Основные команды

| Команда | Описание | Пример |
|---------|----------|--------|
| `help` | Показать список доступных команд | `help` |
| `bpm` | Просмотр или изменение BPM | `bpm 140` |
| `play` | Запуск воспроизведения | `play` |
| `stop` | Остановка воспроизведения | `stop` |

## Параметры инструментов

### Bass (Бас)
- `bass.cutoff <freq>` - частота среза фильтра (Гц)
- `bass.res <q>` - резонанс фильтра (0-10)

### Drums (Ударные)
- `drums.kick.pulses <n>` - количество ударов бочки в сетке (0-16)
- `drums.snare.pulses <n>` - количество ударов малого в сетке (0-16)
- `drums.hihat.pulses <n>` - количество ударов хэта в сетке (0-16)

### Pads (Пэды)
- `pads.bright <val>` - яркость/яркость пэдов (0-1)

## Расширенные функции

### WASM Node (DSP Sandbox)
Через терминал можно управлять узлами WASM в Node Editor (в разработке).

### История команд
Используйте клавиши **Стрелка Вверх** и **Стрелка Вниз** для навигации по истории введенных команд.
