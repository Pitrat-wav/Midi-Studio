# 🤖 AI Guide: Нейросети в Studio Elite

В Studio Elite интегрированы передовые AI технологии для управления и креатива.

## 1. AI Texture Lab (Генерация Текстур)
Создавайте уникальный дизайн для своих инструментов с помощью Stable Diffusion.

### Как использовать:
1.  Нажмите кнопку **🪄 (Magic Wand)** в меню.
2.  Введите **Hugging Face API Key** (можно получить бесплатно на huggingface.co).
3.  Введите промпт, например: *"Liquid gold metal with glowing circuits"*.
4.  Нажмите **Generate**.
5.  Текстура автоматически применится к планетам и инструментам.

## 2. Spatial Hand Tracking (Управление Руками)
Управляйте звуком жестами в воздухе без контроллера.

### Технология
Используется **MediaPipe Hands** для распознавания скелета руки через веб-камеру.

### Жесты
- **Сведение пальцев (Pinch)**: Захват ручки/слайдера.
- **Ладонь вверх/вниз**: Управление фильтром (Cutoff).
- **Движение влево/вправо**: Панорамирование.

*Примечание: Требуется хорошее освещение и разрешение камеры.*
