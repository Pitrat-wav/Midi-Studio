# Руководство по AI Vision и Motion Capture

## Введение
Telegram Midi предоставляет экспериментальную поддержку компьютерного зрения (Computer Vision) для управления визуальной сценой и звуком с помощью движений тела.

## Режим: SKELETON FLOW

Этот режим визуализации использует библиотеку **MediaPipe Pose** для захвата движений пользователя через веб-камеру в реальном времени.

### Как включить
1. Перейдите в режим визуализатора (через `Visualizer View` в меню или на контроллере).
2. Выберите 4-й пресет (Skeleton Flow).
3. Разрешите браузеру доступ к камере.

### Возможности
*   **Скелет**: Отображение 33 ключевых точек вашего тела в виде неоновых линий.
*   **Энергия**: Яркость скелета пульсирует в такт музыке (Kick/Bass).
*   **Магия Рук**: Вокруг кистей рук генерируются частицы, реагирующие на звук.

## Архитектура Neural Engine

Система построена на базе `@mediapipe/tasks-vision` и оптимизирована для работы в браузере.

### Компоненты
1.  **usePoseTracking**: Хук, управляющий жизненным циклом модели.
    *   Загружает WASM-бинарники MediaPipe.
    *   Создает скрытый элемент `<video>`.
    *   Запускает цикл распознавания (`requestAnimationFrame`).
    *   Обновляет `visualStore` (poseData, visibility).

2.  **Performance**:
    *   Модель: `pose_landmarker_lite` (Float16) для максимальной скорости.
    *   Делегат: `GPU` (WebGPGPU) ускорение.
    *   Троттлинг: Распознавание работает только когда активен 4-й визуализатор.

### Приватность
Никакие видеоданные не отправляются на сервер. Вся обработка происходит локально в вашем браузере.
