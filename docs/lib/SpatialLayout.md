# SpatialLayout.ts

### 1. Глобальное Описание (File Overview)
* **Назначение:** Конфигурационный файл, определяющий пространственную архитектуру 3D-студии. Содержит координаты всех инструментов, их регуляторов (Knobs) и предустановки камер для фокусировки.
* **Связи:** 
    * Используется в: `WebGLScene.tsx`, `CameraController.tsx`, `GenerativeKnob3D.tsx`.

### 2. Интерфейс и Константы (Spatial Elements)

* **Константа: `SPATIAL_LAYOUT`**
    * **Где находится:** Строки 25-127.
    * **Логика:** Объект, где ключом является тип инструмента, а значением — его позиция в сцене, настройки камеры (position, lookAt, fov) и карта позиций для каждого регулятора.

### 3. Логика и Математика (Logic & Math)

* **Функция: `getControlWorldPosition(instrument, controlName)`**
* **Что делает:** Возвращает вектор `THREE.Vector3` с мировыми координатами конкретной ручки управления.
* **Математическая модель:**
    Прямой маппинг из словаря координат.
    $$ Pos(I, C) = \text{Layout}[I].\text{controls}[C] $$

* **Построчный разбор примеров позиций:**
    * `drums`: Центр координат `[0, 0, 0]`.
    * `bass`: Левое крыло студии `[-25, 0, 0]`.
    * `harmony`: Правое крыло студии `[25, 0, 0]`.
    * `drone`: Находится "в небе" `[0, 20, -10]`.

* **Математика Камеры (FOV):**
    Использует разные углы обзора (Field of View) для разных инструментов:
    * Harmony: 55 градусов (более сфокусированный вид).
    * Drone: 70 градусов (широкий панорамный вид).
