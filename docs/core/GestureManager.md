# 📘 АУДИТ: GestureManager.ts

## 1. Глобальное Описание

### Назначение
`GestureManager.ts` (реализованный через `useGestureStore`) — это интеллектуальный слой интерпретации пользовательского ввода. Его главная задача — отличать намерения пользователя в сложном 3D-пространстве. Он разделяет обычное вращение камеры (Swipe) от изменения параметров инструментов (Drag), распознает долгие нажатия для вызова радиального меню (Long Press) и обрабатывает специфические взаимодействия для Telegram WebApp (Haptic Feedback).

### Ключевые Подсистемы
- **Gesture Recognizer:** Определение типа жеста (`swipe`, `drag`, `hold`, `tap`) на основе дистанции и времени.
- **Edge Detector:** Распознавание свайпов от краев экрана для навигации между модулями студии.
- **Radial Menu Trigger:** Логика активации контекстного меню при удержании пальца/курсора на объекте.
- **Haptic Integration:** Виброотклик для мобильных устройств при успешном распознавании жеста.

## 2. Логика Распознавания (State Machine)

**Строки: 42-114**

Процесс обработки инпута разделен на три фазы:
1. **onStart:** Фиксация начальной позиции и времени. Определение, началось ли движение от края экрана (`threshold = 60px`). Запуск таймера для `hold` (600мс).
2. **onMove:** Расчет евклидова расстояния от точки старта. Если пройдено более 20px, система фиксирует "активный жест". Если под курсором был 3D-объект (`targetPosition`), выбирается режим `drag`, иначе — `swipe` (камера).
3. **onEnd:** Определение коротких нажатий (`tap`) по длительности менее 200мс. Сброс состояний.

## 3. Математика и Технические Решения

### 3.1 Евклидово расстояние (Movement Threshold)

**Строки: 78-79**

Для отсечения случайного дрожания рук используется расчет дистанции перемещения:
$$Distance = \sqrt{(x_{curr} - x_{start})^2 + (y_{curr} - y_{start})^2}$$
Жест считается начатым только если $Distance > 20px$.

### 3.2 Логика Edge Swipe

**Строки: 43-48**

Система делит экран на "чувствительные зоны" для навигации. Если координата нажатия попадает в полосу 60px от любого края, устанавливается флаг `isEdgeSwipe`. Это позволяет реализовать переключение между Mixer, Sequencer и Synth простым свайпом от края, как в нативных мобильных приложениях.
