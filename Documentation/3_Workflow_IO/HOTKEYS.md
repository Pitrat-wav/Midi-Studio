# 3.2 Global Hotkeys & Controls

Navigation and interaction in the 3D studio are optimized for keyboard shortcuts.

## Scene Navigation
- **`WASD`**: Local movement within the studio.
- **`Mouse Drag`**: Orbit camera around focused instrument.
- **`Scroll`**: Zoom in/out.

## Instrument Controls
- **`1`**: Focus on TR-808.
- **`2`**: Focus on TR-909.
- **`3`**: Focus on modular synth array.
- **`4`**: Focus on Sampler.
- **`ESC`**: Reset camera to studio overview.

## Special Functions
- **`Spacebar`**: Global Play/Stop toggle.
- **`?` (Question Mark)**: Toggle the interactive Info Overlay (spec sheets and control guide) when an instrument is focused.
- **`Backspace`**: Clear current pattern for focused instrument.

## GUI Controls
- **`H`**: Hide/Show 2D UI overlay (HUD).
- **`[` / `]`**: Decrease/Increase global BPM.
