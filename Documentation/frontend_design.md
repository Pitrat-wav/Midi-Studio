# Дизайн и UX (DESIGN_SYSTEM.md)

## Концепция визуального стиля
Стиль проекта — **Premium Glassmorphism**. Это сочетание прозрачных поверхностей, глубокого размытия (blur) и ярких акцентов, создающее ощущение современного и "дорогого" инструмента в стиле Apple/Telegram.

## Дизайн-токены (index.css)

### Цвета (Telegram Native)
Приложение полностью интегрировано с палитрой Telegram через переменные:
- `--tg-theme-bg-color`: Основной фон.
- `--tg-theme-button-color`: Акцентный цвет для кнопок и активных шагов.
- `--tg-theme-hint-color`: Вспомогательный текст.

### Сетка и Отступы
Используется системный подход к пространствам (`--space-unit: 4px`):
- `var(--space-s)`: 8px (мелкие детали).
- `var(--space-m)`: 16px (стандартные отступы).
- `var(--space-l)`: 24px (расстояние между секциями).

### Скругления (Radii)
- `var(--radius-m)`: 12px (малые кнопки, инпуты).
- `var(--radius-l)`: 16px (карточки, модалки).
- `var(--radius-xl)`: 24px (большие приветственные элементы).

## Мобильная оптимизация (Safe Areas)
Приложение учитывает особенности современных экранов:
- **Header**: Использует `padding-top: env(safe-area-inset-top)` для работы под "челкой".
- **Bottom Navigation**: Использует `padding-bottom: env(safe-area-inset-bottom)` для корректного отображения над домашним баром iOS.
- **Тач-таргеты**: Минимальная высота интерактивных элементов — `44px` для удобного управления пальцем.

## Визуальные эффекты
- **Blur**: Используется `backdrop-filter: blur(20px)` для создания эффекта матового стекла.
- **Анимации**: 
  - `motion` (Framer Motion) для плавного появления карточек.
  - Масштабирование `0.97` при нажатии на кнопки для тактильного отклика.
- **Синхронизация**: Вспышки текущего шага имеют свечение (glow) в цвет акцента, синхронизированное с битом.

## Шрифты
Используется системный стек шрифтов Apple/San Francisco:
`-apple-system, BlinkMacSystemFont, 'SF Pro Display', 'Segoe UI', Roboto, Helvetica, Arial, sans-serif;`
Упор сделан на жирные начертания (`700`, `800`) для заголовков и статусных сообщений.
