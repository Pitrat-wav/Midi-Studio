# Telegram MIDI Studio — Документация для Агента

Этот каталог содержит полную техническую информацию о проекте, разделенную по ключевым направлениям. Используйте эти файлы для глубокого анализа системы и планирования изменений.

## 📂 Структура документации

### 🏗 [Архитектура и Системная логика (SYSTEM.md)](./SYSTEM.md)
Обзор потоков данных, управления состоянием (Zustand), инициализации аудио-контекста и глобального цикла секвенсора.

### 🔊 [Аудио-движок и Алгоритмы (AUDIO_LOGIC.md)](./AUDIO_LOGIC.md)
Подробное описание синтезаторов (Acid, FM, HarmPoly) и генеративных алгоритмов (Sting, Bjorklund, Turing Machine, Smart Chord).

### ⚛️ [Фронтенд и Компоненты (FRONTEND.md)](./FRONTEND.md)
Структура React-компонентов, интеграция с Telegram WebApp API, управление UI-состоянием и оптимизация рендеринга.

### 🎨 [Дизайн-система и UX (DESIGN_SYSTEM.md)](./DESIGN_SYSTEM.md)
Гайд по стилю Premium Glassmorphism, описание дизайн-токенов (index.css), поддержка мобильных Safe Areas и тач-оптимизация.
