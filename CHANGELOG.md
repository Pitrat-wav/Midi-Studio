# Журнал изменений (Changelog)

## [Версия 1.2.0] - 2026-01-26
### Добавлено
- **Музыкальный Интеллект**: Пэды теперь генерируют гармонические прогрессии (I-IV-V-I) вместо статических аккордов.
- **Ритмический Swing**: Глобальный параметр свинга для "оживления" ритма.
- **Расширенные Барабаны**: Добавлены Open Hi-Hat и Clap с независимыми эвклидовыми секвенсорами.
- **MIDI Экспорт v2**: Полная поддержка новых инструментов и стандарта General MIDI (Open Hat -> 46, Clap -> 39).

## [Версия 1.1.0] - 2026-01-26
### Добавлено
- **Детерминированная генерация баса**: Внедрен механизм "Seed-based Density" в `StingGenerator`. Теперь при изменении плотности (Density) основные ноты паттерна сохраняются.
- **Поддержка нескольких наборов ударных**: Добавлены киты 808 и 909 с возможностью переключения.
- **Индивидуальный контроль ударных**: Добавлены регуляторы тона (Pitch) и затухания (Decay) для каждого канала (Kick, Snare, Hi-Hat).
- **Мастер-микшер**: Панель управления громкостью для всех инструментов (Acid, Drums, Pads).
- **Осциллограф**: Визуализация звуковой волны в реальном времени.
- **Воркараунд для iOS**: Решена проблема "бесшумного режима" (Silent Switch) с помощью воспроизведения скрытого буфера тишины.
- **Вероятность стадий ML-185**: Добавлен параметр Probability для каждой стадии секвенсора.
- **Глобальная гармония**: Синхронизация всех генераторов с выбранной тоникой и ладом.

## [Версия 1.0.0] - 2026-01-26
### Добавлено
- **Гармонический движок**: Выбор тональности и лада (Major, Minor, Dorian и др.) через Tonal.js.
- **Ambient Pads**: Полифонический синтезатор для создания фоновых аккордов, меняющихся каждые 2 такта.
- **Расширенный экспорт MIDI**: Поддержка 4 дорожек (Bass, Drums, Lead, Pads) с сохранением темпа и артикуляции.
- **Умная сетка Snake**: Интеграция сетки нот с глобальной гармонией.

## [Версия 0.2.0] - 2026-01-26
### Добавлено
- **Секвенсор ML-185**: Полная реализация 8 стадий с режимами гейта (Mute, Single, Multi, Hold).
- **Сетка MDD Snake**: Координатный выбор нот 4x4 с паттернами движения (Linear, ZigZag, Spiral, Random).
- **Визуализация эвклидовых ритмов**: Интерактивное отображение паттернов Kick, Snare и Hi-Hat.
- **Обновление Acid Synth**: Добавлена прямоугольная волна (Square) и улучшена логика акцентов.
- **Визуальный плейхед**: Подсветка активных шагов и стадий во всех представлениях.
- **Интеграция с Telegram**: Локализация интерфейса на русский язык и настройка тактильной отдачи (Haptic Feedback).

### Изменено
- **Оптимизация производительности**: Секвенсор теперь не перезапускается при изменении параметров "на лету".
- **Улучшение UX**: Добавлен сброс регуляторов (Knobs) по двойному клику.
- **Дизайн-система**: Улучшены эффекты стекломорфизма (glassmorphism) и соответствие гайдлайнам Telegram.

## [Версия 0.1.0] - Начальный релиз
- Базовый синтез TB-303 и эвклидовы ритмы.
- Настройка проекта как Telegram Mini App.
