    - Убраны металлические свойства (metalness: 0).
    - Максимальная шероховатость (roughness: 1.0) для получения плоского, пиксельного вида.
- **СТАБИЛЬНОСТЬ**: Усилены проверки безопасности во всех компонентах:
    - HandVision3D: добавлена проверка на `joints.length > 0` перед рендерингом скелета.
    - EffectComposer: полностью удалён из рендеринг-пайплайна для предотвращения крашей.
- **ДОКУМЕНТАЦИЯ**: Обновлены комментарии в коде для объяснения архитектурных решений.
