# 🎮 Midi-Studio — Полное руководство по управлению

## Горячие клавиши

### Транспорт

| Клавиша | Действие |
|---------|----------|
| **SPACE** | Play/Stop |
| **P** | Panic (остановить все звуки) |

### Навигация

| Клавиша | Действие |
|---------|----------|
| **1** | Фокус на Drums (открыть контроллер) |
| **2** | Фокус на Bass |
| **3** | Фокус на Harmony |
| **4** | Фокус на Pads |
| **5** | Фокус на Sequencer |
| **6** | Фокус на Drone |
| **7** | Фокус на Master |
| **0** | Открыть Visualizer Shop |
| **Tab** | Переключение вида (3D → Nodes → Live → Arrange) |
| **H** | Скрыть/Показать HUD |
| **`** (backtick) | Terminal overlay |

### Темы и визуализаторы

| Клавиша | Действие |
|---------|----------|
| **B** | Переключение темы фона (циклически) |
| **?** | Help overlay |

### Поиск

| Клавиша | Действие |
|---------|----------|
| **Cmd+K** | Открыть поиск инструментов |
| **/** | Открыть поиск инструментов (альтернатива) |

## 2D Контроллеры инструментов

### 🥁 Drums (клавиша 1)

**Элементы управления:**
- **808/909** — переключение между драм-машинами
- **START/STOP** — запуск/остановка секвенсора
- **Пэды (6 шт)** — выбор инструмента (Kick, Snare, Hi-Hat, Open Hat, Clap, Ride)
- **Level** — уровень громкости инструмента
- **Tune** — настройка высоты тона
- **Step sequencer (16 шагов)** — программирование паттерна

---

### 🎸 Bass (клавиша 2)

**Элементы управления:**
- **ACID/FM** — переключение между движками
- **Cutoff** — частота среза фильтра
- **Resonance** — резонанс фильтра
- **Morph** — морфинг между волнами
- **Distortion** — уровень дисторшна
- **Root/Scale** — глобальная гармония

**FM режим:**
- **Harmonicity** — соотношение частот
- **Mod Index** — индекс модуляции
- **Attack/Decay** — огибающая

---

### 🎹 Harmony (клавиша 3)

**Элементы управления:**
- **OSC1/OSC2/OSC3** — осцилляторы
- **Waveform** — форма волны (sine/saw/square/triangle)
- **Detune** — расстройка
- **Filter** — фильтр (Freq/Q/Type)
- **Complex** — FM/AM модуляция

---

### ☁️ Pads (клавиша 4)

**Элементы управления:**
- **ACTIVE/STANDBY** — включение/выключение
- **Brightness** — яркость тембра
- **Complexity** — сложность спектра
- **Spread** — стерео разброс
- **Modulation** — глубина модуляции
- **Voice Mode** — Poly/Mono/Unison

---

### 🎬 Sequencer (клавиша 5)

**Элементы управления:**
- **Turing Machine** — генеративный секвенсор
- **ML-185** — алгоритмический секвенсор
- **Snake Grid** — змейка паттернов

---

### ☄️ Drone (клавиша 6)

**Элементы управления:**
- **Root** — базовая нота
- **Texture** — тембральная текстура
- **Density** — плотность слоёв
- **Movement** — скорость модуляции

---

### 🎚️ Master (клавиша 7)

**Элементы управления:**
- **PLAY/PAUSE** — транспорт
- **PANIC** — аварийная остановка
- **BPM** — темп (60-200)
- **Channel Mixer** — микшер каналов
- **FX Processor** — эффекты

---

## MIDI контроллеры

### Launch Control XL

Приложение поддерживает Novation Launch Control XL:
- **Faders** — управление параметрами
- **Knobs** — тонкая настройка
- **Pads** — триггеры пэдов

---

## Gamepad

Приложение поддерживает геймпады:
- **A/B/X/Y** — триггеры
- **D-Pad** — навигация
- **Triggers** — параметры

---

## Советы

1. **Быстрая смена инструментов:** Используйте клавиши 1-7
2. **Скрытие интерфейса:** Нажмите H для чистого просмотра 3D сцены
3. **Смена темы:** Нажимайте B для циклического переключения
4. **Паника:** Нажмите P если звуки "зависли"
